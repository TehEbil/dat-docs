# SilverDAT Schnittstellen-Kompendium

## Cursor @Docs Setup
1. GitHub Pages: Settings → Pages → Source: **GitHub Actions**
2. Push auf `main` → automatisches Deployment
3. Cursor: `@Docs` → "Add new doc" → `https://tehehbil.github.io/dat-docs/`

## Lokal
```bash
pip install mkdocs-material
mkdocs serve
```
