# Anmeldung

Beachten Sie den abweichenden Einstiegspunkt für die Anmeldungsfunktionen. Statt unter
dem /rest-Pfad finden sich die Berechtigungs-Funktionen unter dem /admin-Pfad. Der Pfad zur Login-Funktion lautet somit /admin/login, während die Refresh-Funktion auf dem Pfad /admin/refresh zu finden ist.
