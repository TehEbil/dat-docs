# Definitionen

| Begriff | Bedeutung |
| --- | --- |
| Hersteller | Fahrzeughersteller im DAT-Kontext, faktisch die Fahrzeugmarke |
| Modell | Fahrzeuglinie, zum Beispiel "VW Golf Variant", "Toyota Lexus" |
| Workbook | Enthält alle wichtigen Daten eines bestimmten Modells in aktueller Version und allen zurückliegenden Versionen |
| Workbook-Version | Datensatz eines Modells zu einem bestimmten Zeitpunkt |
| Derivat | Unterste Fahrzeughierarchieebene, definiert einen Datensatz innerhalb einer Arbeitsmappe, z.B. "1.4 TSI Entry" |
| Spezifikationen | Merkmale eines Fahrzeugs; der Katalog wird von DAT zur Verfügung gestellt |
| Ausstattung | Ein zusammenhängender Satz bestimmter Fahrzeugmerkmale, entweder als Serie oder Option |
| Option | Ausstattung, die zusätzlich zur Serienausstattung bestellt werden kann; basiert auf demselben Katalog wie die Serienausstattung |
| Optionspaket | Speziell gekennzeichnete Sonderausstattung oder Sonderausstattungssatz |
| Außenfarbe | Bestellbare Fahrzeugfarbe |
| Polster | Bestellbares Fahrzeugpolster |
| Ausstattungsposition | Eine einzelne Serienausstattung, Option, Außenfarbe oder Polster |
| Positionsauswahl | Sich gegenseitig ausschließende Kombination von eingeschlossenen (included) oder erforderlichen (required) Ausstattungspositionen (ODER-Verbindung) |
| Innere Auswahl | ODER-Verbindung von einzelnen Ausstattungspositionen |
| Äußere Auswahl | ODER-Verbindung von Ausstattungspositionsgruppen |
