# VIN-Abfrage

ich kann keine VIN-Abfrage tätigen (VIN-Freigaben) (oder nur bestimmte Hersteller)

Um VIN-Abfragen tätigen zu können, müssen die jeweiligen Hersteller freigegeben werden.
Hierzu sollten Sie vorab mit ihrem DAT-Kundenbetreuer die VIN-Freigaben klären, damit
diese freigeschaltet werden.
