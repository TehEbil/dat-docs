# SD3 Netze

wie komme in das (VW FHD u.s.w.)-Netz ?

Um auf bestimmte Netze (z.B. DSN (Mercedes-Benz), VW u.s.w) zugreifen zu können, müssen
diese lizensiert sein.

Ersehen können Sie das ganz einfach, indem Sie in ihr Dashboard gehen und dementsprechend
die Netz-Kachel sehen.

Des weiteren steuern Sie das Netz mit dem Parameter networkType in das richtige Netz.

Hierzu Infos im Kompendium:

https://www.dat.de/fileadmin/de/support/interface-documentation/MG/SilverDAT\_Schnittstellen-Kompendium\_Mag/#/home/9283/20/21
