# welche Produkte sind lizenziert

wo finde ich eine Liste welche Produkte ich lizensiert habe?

Wenn Sie einen ROOT-User haben, können Sie über die Kachel "Benutzer- und Rechteverwaltung"
im Dashboard in ihrem Portal unter www.dat.de ihre lizensierten Produkte einsehen
(unter Produktvarianten).
