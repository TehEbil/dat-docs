# Reperaturkostenliste

<RepairDecreaseList> ist ein Unter-Element von <Condition> und enthält eine Liste mit Informationen zur Aufwertung eines Fahrzeugs.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| DetailPositionList | DetailPositionList |  |
| RepairDecreaseSum | Decimal | Benutzerdefinierte Reperaturkostensumme (Netto) |
| RepairDecreaseSumGross | Decimal | Benutzerdefinierte Reperaturkostensumme (Brutto) |
| DatRepairDecreaseSum | Decimal | Reperaturkostensumme DAT (Netto) |
| DatRepairDecreaseSumGross | Decimal | Reperaturkostensumme DAT (Brutto) |
