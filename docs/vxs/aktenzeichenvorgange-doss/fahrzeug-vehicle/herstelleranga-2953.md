# Herstellerangaben zum Fahrzeug (VINVehicle)

< VINVehicle> ist ein Unter-Element von <VinResult> und enthält die Herstellerangaben /-codes zum Fahrzeug.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| ManufacturerCarCode | string | Fahrzeugcode des Herstellers |
| ManufacturerEngineCode | string | Motorkennbuchstabe des Herstellers |
| ManufacturerTransmissionCode | string | Getriebebuchstabe des Herstellers |
