# Wartungsdaten (Maintenance)

<Maintenance> ist ein Unter-Element von <Vehicle> und enthält Informationen zur Wartung des Fahrzeugs.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| LastServiceKilometer | integer | Kilometerstand bei letzter Inspektion |
| LastServiceDate | date | Datum der letzten Inspektion |
| NextServiceKilometer | integer | Kilometerstand für nächste Inspektion |
| NextServiceDate | date | Datum für nächste Inspektion |
| NextEmissionCheck | date | Datum für nächste Abgasuntersuchung |
| NextVehicleInspection | date | Datum für nächste Hauptuntersuchung |
| NextSecurityCheck | date | Datum für nächste Sicherheitsprüfung |
