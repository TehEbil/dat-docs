# Aufwertungsliste

<ValueIncreaseList> ist ein Unter-Element von <Condition> und enthält eine Liste mit Informationen zur Aufwertung eines Fahrzeugs.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| DetailPositionList | DetailPositionList |  |
| ValueIncreaseSum | Decimal | Benutzerdefinierte Aufwertungssumme (Netto) |
| ValueIncreaseSumGross | Decimal | Benutzerdefinierte Aufwertungssumme (Brutto) |
| DatValueIncreaseSum | Decimal | Aufwertungssumme DAT (Netto) |
| DatValueIncreaseSumGross | Decimal | Aufwertungssumme DAT (Brutto) |
