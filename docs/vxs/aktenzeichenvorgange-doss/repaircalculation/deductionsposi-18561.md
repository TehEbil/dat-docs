# DeductionsPositions

Das Element <DeductionsPositions>, welches sich unter "[Kalkulationsergebnisse](kalkulationser-1409.md)" befindet, dient als Elternelement für 1:n <DeductiblePartsPosition>-Elemente. Diese enthalten die Abzugspositionen (NFA, Wertverbesserung, Vorschaden).

Felder:

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| [DeductiblePartsPosition](deductiblepart-18562.md) | complexType, optional  [DeductiblePartsPosition](deductiblepart-18562.md) | Abzugspositionen (NFA, Wertverbesserung, Vorschaden) |
