# RepairPositions

Das Element <RepairPositions>, das sich unter <RepairCalculation> befindet, dient als Elternelement für beliebig viele <[RepairPosition](repairposition-1419.md)>-Elemente. Diese enthalten die vom Benutzer ausgewählten Reparaturpositionen.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| [RepairPosition](repairposition-1419.md) | [RepairPosition[]](repairposition-1419.md) | Eine Liste mit Reparaturpositionen. |
