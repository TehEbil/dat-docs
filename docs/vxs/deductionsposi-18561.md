# DeductionsPositions

Das Element <DeductionsPositions>, welches sich unter "[Kalkulationsergebnisse](#showid/1409 "Kalkulationsergebnisse CalcResult")" befindet, dient als Elternelement für 1:n <DeductiblePartsPosition>-Elemente. Diese enthalten die Abzugspositionen (NFA, Wertverbesserung, Vorschaden).

Felder:

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| [DeductiblePartsPosition](#showid/18562 "DeductiblePartsPosition") | complexType, optional  [DeductiblePartsPosition](#showid/18562 "DeductiblePartsPosition") | Abzugspositionen (NFA, Wertverbesserung, Vorschaden) |
