# SettingsParameter - SparePartSettings

Ersatzteile-Einstellungen:

| Element | Typ | Name | Key | Value | Level | Beschreibung |
| --- | --- | --- | --- | --- | --- | --- |
| <SettingsParameter> | name = String  key = String  value = Boolean  level = String | SparePartSettings | SHOW\_FORD\_FINIS\_NUMBER | [true | false]    true: Es wird die Ford-Finis-Nummer angezeigt  false: Es wird keine Ford-Finis-Nummer angezeigt | company | Ersatzteilnummer statt Ford-Finis-Nummer anzeigen |
| <SettingsParameter> | name = String  key = String  value = Boolean  level = String | SparePartSettings | SPARE\_PART\_INHERIT\_DISCOUNTS | [true | false]    true: NfA und Rabatte auch auf zugesteuerte Teile angewendet  false: Auftragsbezogene Kalkulationsfaktoren werden berücksichtigt | company | NfA und Rabatte auch auf zugesteuerte Teile anwenden |
