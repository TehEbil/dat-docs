# Attachments

Das Element <Attachments>, das sich unter <Dossier> befindet, dient als Elternelement für eine beliebige Anzahl von <[Attachment](attachment-26207.md)>-Elementen. Diese enthalten Informationen über die angehängten Dokumente für einen
bestimmten Vorgang.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| [Attachment](attachment-26207.md) | [Attachment](attachment-26207.md)[] | Enthält eine Liste mit Dokumenten. |
