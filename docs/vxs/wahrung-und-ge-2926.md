# Währung und Geldbeträge

Soweit nicht anders angegeben, sind Geldbeträge immer netto. Die Mehrwertsteuerstätze
sind im Element „VAT" enthalten (siehe ), die Währung im Element „Currency" unter „Dossier". Wo dadurch Rundungsprobleme entstehen, gibt es üblicherweise zwei getrennte Felder
für Brutto („...Gross") und Netto („...Net").
