# DentPositionsProtocolEntry

Felder

| Element | Typ | Beschreibung |
| --- | --- | --- |
| DATProcessId | Integer, optional | Die DVN zu dem Hinweis |
| RepairType | String, optional | Art der Reparatur (RC, z.B. replace) |
| Description | String, optional | Beschreibung für den Hinweis (z.B. Bezeichnung des Ersatzteils) |
| CalculationMethod | String, optional | Hagelkalkulationsmethode (z.B. BVAT) |
| CalculationMethodDescription | String, optional | Beschreibung der Hagelkalkulationsmethode (z.B. Methode: Bundesverband Ausbeultechnik (BVAT), Drücken |
| DentsSize | String, optional | Dellengröße |
| DentsCount | String, optional | Anzahl der Dellen |
