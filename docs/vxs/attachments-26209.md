# Attachments

Das Element <Attachments>, das sich unter <Dossier> befindet, dient als Elternelement für eine beliebige Anzahl von <[Attachment](#showid/26207 "Attachment")>-Elementen. Diese enthalten Informationen über die angehängten Dokumente für einen
bestimmten Vorgang.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| [Attachment](#showid/26207 "Attachment") | [Attachment](#showid/26207 "Attachment")[] | Enthält eine Liste mit Dokumenten. |
