# Weitere Reparaturfaktoren (OtherRepairFactors)

Im Element <OtherRepairFactors>, das sich unter den in "[Kalkulationsergebnisse](#showid/1409 "Kalkulationsergebnisse CalcResult")" aufgeführten Elementen befindet, sind weitere pauschale Reparaturfaktoren enthalten.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| LacquerMaterialPerc | decimal | Lackmaterialanteil in Prozent |
| LacquerMaterialCosts | decimal | Lackmaterialanteil (Betrag) |
| SmallParts | decimal | Kleinteile |
