# Auszeichnung (Labelling)

Das Element <Labelling> ist ein Unterelement von <Dossier>, <Sale> und <Admission>und enthält die Preisdetails für Neufahrzeuge.

| Name | Datentyp | Beschreibung | Schreibweise |
| --- | --- | --- | --- |
| OriginalPricewithEquipmentNet | Decimal | Listenneupreis inkl. werkseitiger Ausstattung (Netto) |  |
| OriginalPricewithEquipmentGross | Decimal | Listenneupreis inkl. werkseitiger Ausstattung (Brutto) |  |
| SalesDiscountPerc | Decimal | Nachlass in Prozent |  |
| SalesDiscountNet | Decimal | Nachlass in Euro (Netto) |  |
| SalesDiscountGross | Decimal | Nachlass in Euro (Brutto) |  |
| SubtotalNet | Decimal | Zwischensumme (Netto) |  |
| SubtotalGross | Decimal | Zwischensumme (Brutto) |  |
| AdditionalEquipmentValueNet | Decimal | Wert der Zusatzausstattungen in Euro (Netto) |  |
| AdditionalEquipmentValueGross | Decimal | Wert der Zusatzausstattungen in Euro (Brutto) |  |
| TransferCostsNet | Decimal | Überführungskosten in Euro (Netto) |  |
| TransferCostsGross | Decimal | Überführungskosten in Euro (Brutto) |  |
| IncludingTransferCosts | Boolean | Überführungskosten inklusive |  |
| IncidentalCostsNet | Decimal | Zulassungs-/Nebenkosten in Euro (Netto) |  |
| IncidentalCostsGross | Decimal | Zulassungs-/Nebenkosten in Euro (Brutto) |  |
| IncludingIncidentalCosts | Boolean | Zulassungs-/Nebenkosten inklusive |  |
| SalesPriceNet | Decimal | Verkaufspreis (Netto) |  |
| SalesPriceGross | Decimal | Verkaufspreis (Brutto) |  |
| DisplayPriceNet | Decimal | Auszeichnungspreis (Netto) |  |
| DisplayPriceGross | Decimal | Auszeichnungspreis (Brutto) |  |
| DeviatingMarketPlacePrice | Boolean | abweichender Börsenpreis |  |
| MinimumSalesPriceNet | Decimal | Mindest-Verkaufspreis (Netto) |  |
| MinimumSalesPriceGross | Decimal | Mindest-Verkaufspreis (Brutto) |  |
| ResellerPriceNet | Decimal | Wiederverkäuferpreis (Netto) |  |
| ResellerPriceGross | Decimal | Wiederverkäuferpreis (Brutto) |  |
| DisplayPriceMarketPlaceNet | Decimal | Auszeichnungspreis Börse (Netto) |  |
