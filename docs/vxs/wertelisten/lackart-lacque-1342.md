# Lackart (LacquerType)

LacquerType aus EquipmentPosition

| Wert | Beschreibung |
| --- | --- |
| metallic | Metallic |
| special | Sonderlackierung (z.B. Effektlackierungen) |
| uni | Einfarbig |

LacquerType aus RepairParameters und calculationWage

Gilt nur für LacquerMethod=EURO\_LACQUER und locale=DE. Für LacquerMethod=MANUFACTURER\_SPECIFIC gelten die Farbcodes des jeweiligen Herstellers. Bei der Übergabe als Requestparameter
wird der Wert eingegeben, als Rückgabeelement wird die Beschreibung ausgegeben.
