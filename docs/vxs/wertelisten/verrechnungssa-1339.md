# Verrechnungssatz calculationWage

| Attribut 'type' | Beschreibung |
| --- | --- |
| VRO\_REPAIRWAGE\_ELECTRIC | Elektrik |
| VRO\_REPAIRWAGE\_MECHANIC | Mechanik |
| VRO\_REPAIRWAGE\_BODY | Karosserie |
| lacquer | Lack |
| dent | Dellen drücken |
| lacquerType | [Lackart](lackart-lacque-1342.md) |

Der Wert für das Attribut level liegt zwischen 1 und 4, außer bei den Typen dent und lacquer.

Möchten Sie einen Level nicht angeben, dann lassen Sie den Wert zwischen der Definition
der calculationWage Elemente des jeweiligen Levels leer (auch eine „0" ist ein Wert).
