# RepairPositions

Das Element <RepairPositions>, das sich unter <RepairCalculation> befindet, dient als Elternelement für beliebig viele <[RepairPosition](#showid/1419 "RepairPosition")>-Elemente. Diese enthalten die vom Benutzer ausgewählten Reparaturpositionen.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| [RepairPosition](#showid/1419 "RepairPosition") | [RepairPosition[]](#showid/1419 "RepairPosition") | Eine Liste mit Reparaturpositionen. |
