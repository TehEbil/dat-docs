# Abwertungsliste

<ValueDecreaseList> ist ein Unter-Element von <Condition> und enthält eine Liste mit Informationen zur Aufwertung eines Fahrzeugs.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| DetailPositionList | DetailPositionList |  |
| ValueDecreaseSum | Decimal | Benutzerdefinierte Abwertungssumme (Netto) |
| ValueDecreaseSumGross | Decimal | Benutzerdefinierte Abwertungssumme (Brutto) |
| DatValueDecreaseSum | Decimal | Abwertungssumme DAT (Netto) |
| DatValueDecreaseSumGross | Decimal | Abwertungssumme DAT (Brutto) |
