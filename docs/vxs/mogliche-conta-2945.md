# Mögliche Container zu DAT €uropa-Code  (VINContainers)

##### Mögliche Container zu DAT €uropa-Code (VINContainers)

<VINContainers> ist ein Unter-Element von <VINECode> und enthält die möglichen Container zum DAT €uropa-Code®.

Felder:

| Element | Typ | Beschreibung |
| --- | --- | --- |
| Container | long | Container-Schlüssel aus dem VIN-Ergebnis |
| VehicleConstructionTime | long | Bauzeit des Fahrzeugs |
| VehicleTypeKey | Integer | Fahrzeugart Schlüsselwert |
| ManufacturerKey | Integer | Hersteller Schlüsselwert |
| VehicleMainTypeKey | Integer | Haupttyp Schlüsselwert |
| VehicleSubTypeKey | Integer | Untertyp Schlüsselwert |
