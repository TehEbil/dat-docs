# Vertragsdaten

<ContractData> ist ein Unter-Element von <Dossier> und enthält die Informationen zu den Vertragsdaten.

Element vxs:ContractData

| Elemente | Typ | Beschreibung |
| --- | --- | --- |
| ContractNumber | String | Vertragsnummer |
| CustomerNumber | String | Kundennummer |
| ContractStartingDate | Date | Vertragsbeginndatum |
| MileageAtContractBegin | Integer | Laufleistung zum Vertragsbeginn in km |
| ContractDurationInMonths | Integer | Vertragslaufzeit in Monaten |
| ContractMileage | Integer | Vertragslaufzeit in km |
| ContractEndDate | Date | Vertragsendedatum |
| MileageAtContractEnd | Integer | Laufleistung zum Vertragsende in km |
