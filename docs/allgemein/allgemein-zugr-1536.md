# Allgemein: Zugriff auf DAT-Online-Kundenumgebung

Zur Performanceverbesserung vor allem beim Zugriff aus dem Ausland auf die Internetumgebung,
verwenden wir einen CDN Provider für die Online Kundenumgebung. Bitte beachten Sie
die Voraussetzungen für einen fehlerfreien Zugriff. Ihr Client muss mindestens den
Anforderungen von HTTP/1.1 entsprechen.
