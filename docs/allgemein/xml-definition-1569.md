# XML Definition

XML (Extensible Markup Language) ist der Industrie-Standard zur Definition von Dokumenten.

Ein Element besteht aus einem öffnenden Tag '<ElementName>', dem Elementinhalt und einem schließenden Tag '</ElementName>'.

Als Elementinhalt kommen die Daten oder weitere Elemente in Frage. Jedes XML-Dokument ist in einer Baumstruktur mit einem eindeutigen Wurzelelement angeordnet.
XML unterscheidet Groß- und Kleinschreibung.

Jedes XML-Dokument besteht aus einem [Prolog](#showid/1571 "Prolog"), dem [Wurzelelement (und seinen Inhalten) sowie möglichen Kommentaren](#showid/1573 "XML-Aufbau").
