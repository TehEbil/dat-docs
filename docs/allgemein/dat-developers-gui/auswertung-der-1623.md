# Auswertung der DVN-Beschreibungstexte

###### Auswertung der DVN-Beschreibungstexte Im Regelfall werden die Beschreibungstexte aus <metadata><lang> eindeutig mit der darunterliegenden DVN verknüpft und dargestellt. Manchmal ist in Beschreibungstexten das Symbol "@". Diese Beschreibungstexte müssen mit einer besonderen Logik behandelt werden, damit sie korrekt dargestellt werden. Die Beschreibungstexte links und rechts des Symbols "@" stellen jeweils unabhängige Beschreibungstexte zu zwei unterschiedlichen DVNs dar; der Beschreibungstext links von Symbol "@" muss deshalb mit der linken DVN <dvnLeft> und der Beschreibungstext rechts von Symbol "@" mit der rechten DVN <dvnRight> verknüpft und dargestellt werden. Das folgende Beispiel soll die innere Verknüpfung zwischen Beschreibungstext und DVN näher erläutern: ``` <metadata> <dat:objectInfo position="72">   <dat:title uebId="3806237" stnr="24659611">     <dat:langs>       <dat:lang languageCode="D" countryCode="D" stnr="24659611">Thermostat @ Dichtung Thermostat       </dat:lang>       <dat:lang languageCode="E" countryCode="E" stnr="24660272">Termostato @ Junta Termostato       </dat:lang>       <dat:lang languageCode="ENG" countryCode="D" stnr="24660273">thermostat @ gasket thermostat       </dat:lang>       <dat:lang languageCode="F" countryCode="F" stnr="24660274">Thermostat @ Joint Thermostat       </dat:lang>       <dat:lang languageCode="GR" countryCode="GR" stnr="24660275">Θερμοστάτης @ Μόνωση Θερμοστάτης       </dat:lang>       <dat:lang languageCode="H" countryCode="H" stnr="24660276">termosztát @ tömítés termosztát       </dat:lang>       <dat:lang languageCode="I" countryCode="I" stnr="24660277">termostato @ guarnizione termostato       </dat:lang>       <dat:lang languageCode="RO" countryCode="RO" stnr="24660280">termostat @ garnitura termostat       </dat:lang>       <dat:lang languageCode="RUS" countryCode="RUS" stnr="24660281">Термостат @ Прокладка Термостат       </dat:lang>       <dat:lang languageCode="TR" countryCode="TR" stnr="26123955">termostat @ düz conta termostat       </dat:lang>       <dat:lang languageCode="CZ" countryCode="CZ" stnr="28459879">termostat @ těsnění termostat       </dat:lang>       <dat:lang languageCode="SK" countryCode="SK" stnr="30552591">termostat @ tesnenie termostat       </dat:lang>     </dat:langs>   </dat:title>   <dat:dvnGroup>0</dat:dvnGroup>   <dat:kzSeite>e,h</dat:kzSeite>   <dat:dvnLeft>     <dat:dvn>83730</dat:dvn>     <dat:etBauartOptKz>0</dat:etBauartOptKz>     <dat:rcs>       <dat:rc>E</dat:rc>     </dat:rcs>   </dat:dvnLeft>   <dat:dvnRight>     <dat:dvn>83740</dat:dvn>     <dat:etBauartOptKz>0</dat:etBauartOptKz>     <dat:rcs>       <dat:rc>E</dat:rc>     </dat:rcs>   </dat:dvnRight> </dat:objectInfo> </metadata> ``` In dem beispielhaften Auszug aus einer SVG-Datei findet man im Element <metadata><lang> den Beschreibungstext "Thermostat @ Dichtung Thermostat". Außerdem gibt es einen Wert "83730" für <dvnLeft> und einen Wert "83740" für <dvnRight>.
