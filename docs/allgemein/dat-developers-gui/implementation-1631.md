# Implementation

Es gibt verschiedene Möglichkeiten, eine SVG-Datei in einem Browser darzustellen. In den folgenden Kapiteln stellen wir Ihnen
unsere besondere Empfehlung einer Implementierung vor, [Einbetten einer SVG-Datei als Objekt](einbetten-eine-1633.md).

Außerdem stellen wir Ihnen eine weitere Möglichkeit der Implementierung vor, [Konversion der SVG-Datei zu Flash](konversion-der-1637.md), die wir wegen der etwas umständlichen Handhabung allerdings nicht empfehlen können.
Für die Verwendung in älteren Browsern stellt sie aber vielleicht doch eine Option
dar.
