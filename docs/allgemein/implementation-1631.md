# Implementation

Es gibt verschiedene Möglichkeiten, eine SVG-Datei in einem Browser darzustellen. In den folgenden Kapiteln stellen wir Ihnen
unsere besondere Empfehlung einer Implementierung vor, [Einbetten einer SVG-Datei als Objekt](#showid/1633 "Einbetten einer SVG-Datei als Objekt").

Außerdem stellen wir Ihnen eine weitere Möglichkeit der Implementierung vor, [Konversion der SVG-Datei zu Flash](#showid/1637 "Konversion der SVG-Datei zu Flash-Image"), die wir wegen der etwas umständlichen Handhabung allerdings nicht empfehlen können.
Für die Verwendung in älteren Browsern stellt sie aber vielleicht doch eine Option
dar.
