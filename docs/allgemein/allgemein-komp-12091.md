# Allgemein: Kompatibilität

Bitte beachten Sie die nachfolgenden Hinweise für einen fehlerfreien Zugriff.

- Datenaustausch im XML-Format
- Schnittstellenanpassungen werden – sofern möglich - abwärtskompatibel umgesetzt
- Inkompatible Änderungen werden mindestens 6 Monate vorher in einem Newsletter angekündigt
- Das VXS wird mit jedem neuen Produkt und mit jeder neuen Funktion stetig erweitert

Verwenden Sie immer einen XML-Parser und schalten Sie die Schema-Validierung aus
