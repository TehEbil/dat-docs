# Prolog

Der Prolog besteht aus der XML-Deklaration. Sie gibt die Version des XML-Dokumentes
sowie den Zeichensatz an.

```
<?xml version="1.0" encoding="utf-8"?> 
```

Dann folgen in der Regel Namensräume sowie Verarbeitungsanweisungen und Kommentare.
