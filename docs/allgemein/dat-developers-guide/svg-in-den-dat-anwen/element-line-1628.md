# Element line

###### Element line Line-Elemente stellen Liniensegmente dar, die an einem definierten Punkt beginnen und an einem definierten Punkt enden. Sie werden in Verbindung mit Path-Elementen benutzt. Beispiel ``` <line stroke-width="0.12" x1="47.192" y1="35.121" x2="46.966" y2="34.991" /> ```
