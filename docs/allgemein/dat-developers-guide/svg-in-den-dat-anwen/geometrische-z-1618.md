# Geometrische Zeichenelemente und -pfade

Eine SVG-Datei kann die folgenden Basis-Zeichenelemente benutzen, um die graphischen Teile
(DVN) darzustellen:

- Polygon: Diese Elemente definieren einen Verbund von geraden Liniensegmenten. Im Regelfall
  stellen sie geschlossene Formen dar. Polygon-Elemente werden hauptsächlich in Pixelgraphiken zum Zeichnen benutzt.
- Polyline: Diese Elemente definieren einen Verbund von geraden Liniensegmenten. Im Regelfall
  stellen sie offene Formen dar. Sie werden in Verbindung mit Polygon-Elementen benutzt.
- Path: Diese Elemente definieren den Umriß einer graphischen Gestalt. Mathematisch gesehen
  sind Path-Elemente ein Zusammenschluß von Polygon, Polyline und Line-Elementen.
- Line: Diese Elemente stellen Liniensegmente dar, die an einem definierten Punkt beginnen
  und an einem definierten Punkt enden. Sie werden in Verbindung mit Path-Elementen benutzt.

In den nachfolgenden Kapiteln werden diese Basis-Zeichenelemente weiter beschrieben.
