# Authentifizierung der SilverDAT Webservices

Der Zugriff auf die Webserviceoperationen findet immer innerhalb einer virtuellen
Session statt. Zur Authentifizierung wird die Transaction-based Methode verwendet.
Bei der Transaction-based Methode werden die Elemente der Authentifizierung bei jedem
Zugriff im HTTP-Header mitgegeben.
