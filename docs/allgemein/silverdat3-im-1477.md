# SilverDAT®3 im Einzelnen

SilverDAT® 3 ist ein leistungsfähiges und umfassendes Werkzeug für Automobilbetriebe. Es basiert
auf den Erfahrungen und Erkenntnissen von über 25 Jahren Produktentwicklung bei SilverDAT II und über 85 Jahren DAT Know-how. SilverDAT® 3 dient der professionellen Identifikation von Fahrzeugen, der Wertermittlung z.B.
bei der Inzahlungnahme, der Kalkulation von Reparaturkosten, der Erstellung von Restwertprognosen,
und es unterstützt die Anwender beim Schadenmanagement.

Der Leistungsumfang von SilverDAT® 3 beinhaltet die Funktionen von SilverDAT II. Darüber hinaus sind weitere nützliche
Features hinzugefügt:

- Neue Funktionen: spezielle Kennzeichnung von Materialien, AZT-Lack, Aufbereitungs-und
  IFL-Positionen, Beispiel-Fahrzeugbilder, webScan, neue Kennzahlenauswertung, neue
  Formulare und Vorlagenverwaltung u.v.m.
- Professionelles Schadenmanagement: Digitale Schadenakte und Kommunikation mit Versicherungen
  sowie allen weiteren Beteiligten am Schadenfall inklusive Feedback, Chat-Fenster und
  Statusanzeige.
- Flexible Software: An SilverDAT® 3 können weitere Schadennetze von Automobilherstellern, Versicherungen, Flottenbetreibern
  etc. angebunden werden.
- Neue Technologie: browserbasiertes System der modernsten Softwaretechnologie für einfachen,
  sicheren Zugriff‑ mit zahlreichen individuellen Einstellungsmöglichkeiten.
- Mobil arbeiten: SilverDAT® 3 läuft auf mobilen Endgeräten und ermöglicht dadurch ein dezentrales Arbeiten, z.B.
  bei der Schadenerfassung direkt am Fahrzeug.
- Modulare Struktur aus einer Hand: Fahrzeugbewertung, Reparaturkostenkalkulation, Schadenmanagement
  - auf den Bedarf zugeschnitten, läuft SilverDAT® 3 unabhängig vom Betriebssystem.
- Moderne Optik: zeitgemäßes Design mit intuitiver Benutzerführung.
- Höhere Effektivität: Ein Reitersystem und Icons führen den Nutzer durch die Prozesse.
  Dadurch werden Fehler vermieden und der Schulungsbedarf reduziert.
- Beschleunigung der Prozesse: Arbeiten ohne Medienbruch mit einer einzigen Datenstruktur.
  Das sorgt für schnelleren Zugriff auf die Unterlagen, das mehrfache Erfassen von Daten
  fällt weg.

Nahezu alle neuen Funktionen von SilverDAT® 3 können auch über die Schnittstellen angesprochen werden.
