# Element polyline

###### Element polyline Polyline-Elemente definieren einen Verbund von geraden Liniensegmenten. Im Regelfall stellen sie offene Formen dar. Sie werden in Verbindung mit Polygon-Elementen benutzt. Beispiel ``` <polyline points="35.14306,80.94231 31.74760,82.80500 31.74760,84.15968 31.74760,84.66769  31.74760,86.02237 31.23828,86.36104 31.23828,87.03838 31.57782,87.37706 31.40805,89.40908  30.89873,90.25576 30.72896,92.45712 " fill="none" stroke="#000000" stroke-opacity="1"  stroke-width="0.25" /> ```
