# getData - mögliche Fehlercodes

| Fehlercode | Beschreibung |
| --- | --- |
| [dso-token-01] - token unknown | DSO-Partner: der übergebene dsoToken ist unbekannt.  Beisielsweise wurde ein Token bereits verwendet. |
| [dso-token-02] - token expired | DSO-Partner: der übergebene dsoToken ist älter als erlaubt und somit "abgelaufen". |
| [dso-token-03] - user data invalid | DAT: DAT-Benutzer konnte nicht ermittelt werden. |
| [dso-token-04] - please contact a system administrator | DAT: DSO-Konfiguration ist unvollständig oder enthält Fehler |
| [dso-token-05] - token invalid | DSO-Partner: der übergebene dsoToken ist ungültig. |
| [dso-token-06] - user not authorized | DAT: der Benutzer verfügt nicht über das Recht, Funktionen für den Aufruf des DSO-Partners zu nutzen. |
| [dso-token-07] - token generation failed | DAT: die Erzeugung des tokens ist gescheitert. |
| [dso-token-08] - server error | DAT: DSO-Konfiguration ist unvollständig oder enthält Fehler |
| [dso-token-09] - data providing failed | DAT: DSO-Konfiguration ist unvollständig oder enthält Fehler |
| [dso-token-10] - Resource Not found! | DAT: DSO-Konfiguration ist unvollständig oder enthält Fehler |
| [dso-token-11] - server error! | DAT: DSO-Konfiguration ist unvollständig oder enthält Fehler |
