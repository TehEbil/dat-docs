# High Mobility Plattform

Registrierung

Registrieren Sie sich kostenlos auf der [High-Mobility-Website](https://www.about.high-mobility.com) um Zugriff zur Auto API und den kostenlosen Telematiktesttools zu erhalten. Auf unserer
Plattform können Sie jederzeit Zugangsdaten für die Nutzung der Schnittstelle durch
ihre Anwendung generieren. Legen Sie fest, welche Datenpunkte für Ihr Angebot benötigt
werden und entdecken Sie zusätzliche Hinweise der Community.

Protoyping

Einzigartig bei der Integration von Telematikschnittstellen ist unser [Fahrzeugemulator](https://blog.high-mobility.com/blog/getting-to-grips-with-high-mobilitys-car-emulators). Zum Testen und Erstellen von Prototypen müssen Sie keine Fahrzeuge physisch bewegen.
Mit unserem kostenlosen Emulator bieten wir vielfache Möglichkeiten um Ihr neues Telematikprodukt
beim Entstehungsprozess wachsen zu lassen:

- Visuelle Oberfläche für Einstellung sämtlicher Auto API Werte
- Auslösen von Ereignissen (Unfälle, Wartungsbedarf, Niedriges Batterielevel, ...)
- Testen von herstellerbezogenen Ausnahmefällen (Verbindungsverlust, Fehlermeldungen)
- Erstellen und Abspielen von Simulationen (Gefahrene Routen und Ereignisse)

Go-Live

Mit nur wenigen Klicks kann aus dem Prototyp eine produktive Anwendung werden. Preise
für einzelne Datenpunkte werden direkt pro Hersteller auf der Plattform dargestellt.
Nach unserem standardmäßigen Review geht Ihr neuer Service online und Fahrzeuge lassen
sich digital verknüpfen.
