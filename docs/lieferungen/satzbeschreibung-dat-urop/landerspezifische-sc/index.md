# Länderspezifische Schlüssel

Der DAT-€uropa-Code® ist ein länderübergreifender Schlüssel. Verschiedene Länder haben
eigene Schlüssel definiert, die im jeweiligen Land ebenfalls zur Identifikation von
Fahrzeugen genutzt werden.

- [hsn-und-tsn-de-2841](hsn-und-tsn-de-2841.md)
