# Beschreibung Zusatzelement 3

Weitere technische Informationen

Das Zusatzelement 3 enthält zusätzliche, vorwiegend technische Informationen, die
zum Teil von der Bewertungsbauzeit abhängig sind (Voraussetzung: „ECode 2). Bewertungsbauzeiten
sind der Mai jedes Baujahres.

Es wird ein Satz pro DAT €uropa-Code - Marktindex – Bewertungsbauzeit ausgegeben.

Gibt es zu einem Marktindex keine (gültigen) Bewertungsbauzeiten, wird ein Satz ohne
Bewertungsbauzeit mit den davon unabhängigen Attributen wie PS und Schadstoffklasse
ausgegeben

Der Name des XML-Root-Elements lautet „zusatz3" und der Name eines Satz-Elements lautet
„zusatz3\_s".
