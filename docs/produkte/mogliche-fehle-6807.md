# Mögliche Fehlercodes getVehicleImagesN

| Rückgabecode | Fehlertext | Beschreibung |
| --- | --- | --- |
| wrongValueCode | wrong ECode | Falls kein DAT €uropa-Code® angegeben wurde oder er ein ungültiges Format aufweist. |
| imageAspectNotValid | aspect not valid | Falls kein Aspekt angegeben wurde oder er nicht einen der vorgegebenen Werte (ALL, FRONTVIEW, ANGULARFRONT, SIDEVIEW, REARVIEW, DASHBOARD, ANGULARREAR oder TRUNK) aufweist. |
| imageTypeNotValid | image type not valid | Falls keine Bildart angegeben wurde oder sie nicht den vorgegebenen Wert (PICTURE) aufweist. |
| imagesNotAvailable | no pictures available | Falls zu gegebenem DAT €uropa-Code®, Aspekt und Bildart kein Bild vorhanden ist. |
