# Sonderfeststellungen holen

Mit der Funktion getBasecheckSpecialAssesmentsDetails/getLeasingReturnSpecialAssesmentsDetails /getValuationExpertiseSpecialAssesmentsDetails/getEstimationCertificateSpecialAssesmentsDetails
holen Sie die Sonderfeststellung zu einem Produkt (Grundüberprüfung, Leasingrückgabe,
Bewertungsgutachten oder Schätzurkunde) von dem bestehenden Vorgang.

Parameter   
getBasecheckValueDefinitionDetails  
getLeasingReturnValueDefinitionDetails  
getConditionReportValueDefinitionDetails  
getValuationExpertiseValueDefinitionDetails  
getEstimationCertificateValueDefinitionDetails

| Name | Datentyp | Beschreibung | Schreibweise | Pflicht |
| --- | --- | --- | --- | --- |
| DossierId | Long | Einstiegselement, enthält sämtliche Eingabedaten zum Vorgang |  | x |
