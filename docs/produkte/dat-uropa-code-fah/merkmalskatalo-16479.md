# Merkmalskataloge

Die statische Dokumentation der verfügbaren Merkmalskataloge im SilverDAT Schnittstellen
Kompendium wird durch eine neue Schnittstellenfunktion, getEquipmentAttributeValueLists, ersetzt. Diese neue Funktion ermöglicht den Zugang zu stets aktuellen Merkmalskatalogen.
Zudem können die Merkmalskataloge nun auch für verschiedene Märkte und in den verfügbaren
Sprachen abgerufen werden.

Die Dokumentation der Merkmalskataloge im SilverDAT Schnittstellen Kompendium entfällt
zukünftig.
