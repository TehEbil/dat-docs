# zweiter Aufruf

Hat der Anwender einen Hersteller ausgewählt, kann der Schnittstellenpartner die interaktive
Fahrzeugauswahl durch Setzen des Filters für den Hersteller fortsetzen.

In diesem Beispiel stehen alle für den Zielmarkt Deutschland und Fahrzeugart PKW,
SUV, Transporter verfügbaren Hersteller als "proposals" zur Auswahl zur Verfügung.

- Auswahl des Herstellers BMW.
