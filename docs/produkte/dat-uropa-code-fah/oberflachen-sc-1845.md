# Oberflächen-Schnittstelle

In diesem Kapitel wird die Oberflächen-Schnittstelle der DAT €uropa-Code® Fahrzeugauswahl
vorgestellt. Diese dient mit dem DAT €uropa-Code® als Anwendung zur Identifikation
von Fahrzeugen. Eine weitere optionale Aufgabe ist, die Konfiguration von Sonderausstattungen.
Das erklärt die Einsatzmöglichkeiten des DAT €uropa-Code® sowie der dazugehörigen
Datenlieferungen. Diese Schnittstellen sind reine Datenschnittstellen in Form von
SOAP-basierten Webservices. Es kann aber auch die Oberfläche der DAT €uropa-Code®
Fahrzeugauswahl in Fremdanwendungen integriert werden mit dem Ziel, eine eigene Fahrzeugauswahl
zu installieren. Im folgenden Kapitel geht es um diese Anwendung des DAT €uropa-Code®
als Oberflächen-Schnittstelle zur Fahrzeugauswahl.
