# Allgemeines

Die DAT €uropa-Code® Auswahl ermöglicht die Ermittlung eines DAT €uropa-Code®. Das
heißt Hersteller, Modell, Modellvariante, Motor, Getriebe und Karosserie vom Fahrzeug
werden bestimmt. Der DAT €uropa-Code® kann über Schnittstellen und Webservices auch
für eine eigene Fahrzeugauswahl genutzt werden. Im Gegensatz zur reinen Datenlieferung
in Form von XML- Flatfiles sind hier die entsprechenden Logiken in der Zusammenstellung
der Fahrzeugausstattungen berücksichtigt.
