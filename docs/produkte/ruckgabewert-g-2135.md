# Rückgabewert getNewConctractNumbers

Die Rückgabe erfolgt als Ergebnisliste. Es werden lediglich die Auftrags- bzw. Ordnungsnummern
übertragen. Weitere Daten werden nicht zurückgegeben. Wenn keine nicht-abgerufenen,
versendeten Kalkulationen vorhanden sind, wird ein leerer String zurückgeliefert.
