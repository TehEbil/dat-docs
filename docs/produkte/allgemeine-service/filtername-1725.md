# filterName

Der filterName ist ein von der DAT hinterlegter Name für eine Sammlung von Produktkonfigurationen
inklusive derer Eigenschaften wie z.B. Datenländer, Hersteller usw. sowie optionale
Dienste/Funktionen, wie zum Beispiel die komplette VIN-Lizenzierung.

Wichtig: Die zur Verfügung stehenden Produktkonfigurationen werden von der Hauptkundennummer
generiert/geerbt.
