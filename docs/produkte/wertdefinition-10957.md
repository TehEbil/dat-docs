# Wertdefinitionen holen zu einem Produkt in ExpertPlusPartner

Mit der Funktion getBasecheckValueDefinitionDetails/getLeasingReturnValueDefinitionDetails/getConditionReportValueDefinitionDetails/getValuationExpertiseValueDefinitionDetails/getEstimationCertificateValueDefinitionDetails holen Sie Wertedefinitionsdaten zu einem Produkt (Grundüberprüfung, Leasingrückgabe, Zustandsbericht, Bewertungsgutachten oder Schätzurkunde) von einem bestehenden Vorgang.

Parameter   
getBasecheckValueDefinitionDetails  
getLeasingReturnValueDefinitionDetails  
getConditionReportValueDefinitionDetails  
getValuationExpertiseValueDefinitionDetails  
getEstimationCertificateValueDefinitionDetails

| Name | Datentyp | Beschreibung | Schreibweise | Pflicht |
| --- | --- | --- | --- | --- |
| DossierId | Long | Einstiegselement, enthält sämtliche Eingabedaten zum Vorgang |  | x |
