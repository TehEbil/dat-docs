# Löschen der Bezeichner der Fahrzeugauswahlparameter

Sie löschen eventuell gesetzte Bezeichner der Fahrzeugauswahl mittels der Funktion
changeDossierN. Hierbei werden sowohl die Benutzerwerte als auch die DAT-Defaults gelöscht.

Die nachfolgende Tabelle zeigt die Übersicht der betroffenen Elemente.

Element vxs:Vehicle

| Parameter | Datentyp | Beschreibung | Schreibweise | Pflicht |
| --- | --- | --- | --- | --- |
| vxs:VehicleTypeNameN | String | Frei definierbare Beschreibung für den Fahrzeugartnamen |  |  |
| vxs:ManufacturerName | String | Frei definierbare Beschreibung für den Herstellernamen |  |  |
| vxs:BaseModelName | String | Frei definierbare Beschreibung für den Haupttypnamen |  |  |
| vxs:SubModelName | String | Frei definierbare Beschreibung für den Untertypnamen |  |  |
| vxs:EngineNameManual | String | Frei definierbare Beschreibung für den Motorbezeichnung |  |  |
| vxs:BodyNameManual | String | Frei definierbare Beschreibung für den Karosserie |  |  |
| vxs:GearboxNameManual | String | Frei definierbare Beschreibung für den Getriebe |  |  |
| vxs:EquipmentLineNameManual | String | Frei definierbare Beschreibung für den Ausstattungslinie |  |  |
| vxs:ContainerNameN | String | Frei definierbare Beschreibung für den Marktindex |  |  |
