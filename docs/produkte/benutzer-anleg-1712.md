# Benutzer anlegen oder ändern: createOrUpdateUser

Die Methode aktualisiert die E-Mailadresse des Benutzers.

Bei Kunden, die über den alten Customer-Service angelegt wurden, kann über diese Methode
ein Benutzer zum Kunden angelegt werden.
