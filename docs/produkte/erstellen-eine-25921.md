# Erstellen eines neuen Vorgangs Neufahrzeug, Tageszulassung, Vorführfahrzeug

Für die Erstellung eines Vorgangs für ein Neufahrzeug, eine Tageszulassung oder ein
Vorführfahrzeug können Sie die Funktion createValuationN nutzen. Mittels doValuationInMemoryN können Fahrzeugdaten und der Fahrzeugpreis abgerufen werden.

Generell gilt die Beschreibung zum [Verwalten von Vorgängen](#showid/15005 "Funktionsumfang Vorgangsverwaltung"), ergänzt um die im Folgenden genannten Parameter.
