# fünfter Aufruf

Es sind nur noch zwei Fahrzeuge übrig. Es kann entweder direkt mit einem der beiden
Fahrzeuge weitergearbeitet werden oder über die noch verbleibenden Filteroptionen
eingeschränkt werden.

Bitte nicht vergessen das X-DAT-VS-TOKEN im Request zu verwenden!

- Auswahl des Getriebe, Automatik, mit Steptronic, 8-Stufen.
