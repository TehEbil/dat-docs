# Löschen von manuellen Positionen

Mit der Schnittstellenfunktion deleteAllManualPositions() werden die eigenen, wiederkehrenden "Manuelle Positionen" (Material- und/oder Lohnpositionen)
gelöscht.

Parameter

Es ist keine Parameterübergabe erforderlich. Das Request muss leer abgeschickt werden.

Rückgabe

| Name | Datentyp | Beschreibung |
| --- | --- | --- |
| deleteSuccessful | Boolean | Ist der Wert auf "true" gesetzt, deutet dies darauf hin, dass die hinterlegten manuellen Positionen erfolgreich gelöscht worden sind. |
