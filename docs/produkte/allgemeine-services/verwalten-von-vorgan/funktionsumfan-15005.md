# Funktionsumfang Vorgangsverwaltung

Mit Hilfe der Dossier-Funktionen verwalten Sie Ihre Vorgänge über die Soap Schnittstelle.

Hinweis: Das Produkt SilverDAT 3 valuateFinance benötigt keine Produktvariante. Bei SilverDAT 3 valuateExpert und SilverDAT 3 valuateExpertPlusPartner muss eine Produktvariante angegeben werden.

Übersicht der Funktionen und deren Verfügbarkeit

| Funktion | Beschreibung | SilverDAT 3 valuateFinance | SilverDAT 3 valuateExpert SilverDAT3 valuateExpertPlusPartner |
| --- | --- | --- | --- |
| [createDossierN](erstellen-eine-14999.md) | Erstellen von neuen Vorgängen. | X | X |
| [getDossierN](holen-eines-vo-15007.md) | Abrufen von bestehenden Vorgängen. | X | X |
| [changeDossierN](aendern-eines-14989.md) | Ändern von Vorgängen. | X | X |
| [updateDossierN](aktualisieren-14991.md) | Aktualisieren der Bewertung eines Vorgangs. | X | X |
| [exportDossierN](exportieren-vo-12657.md) | Erstellen von druckfähigen Dokumenten mit Urkundennummer und BAFA Gutachten |  | X |
| [exportDossierN](exportieren-vo-1506.md) | Ausdrucke erstellen | X |  |
| [resetDossier2DefaultN](zurucksetzen-e-15072.md) | Zurücksetzen eines Vorgang auf die DAT Standardeinstellungen. | X | X |
| [resetDossierTechDataN](zurucksetzen-v-25260.md) | Zurücksetzen von technisches Daten eines Vorgangs auf die DAT Standardeinstellung | X | X |
| [deleteDossierN](loschen-eines-15009.md) | Löschen von bestehenden Vorgängen. | X | X |
| [searchDossierListN](suchen-von-vor-15068.md) | Auffinden von bestehenden Vorgängen. | X | X |

Die spezifischen Schnittstellen-Funktionen derSilverDAT 3 valuateFinance, SilverDAT 3 valuateExpert und SilverDAT 3 valuateExpertPlusPartner zum Verwalten der Vorgänge und ihre Beschreibungen rufen Sie über folgende URL's
auf:
