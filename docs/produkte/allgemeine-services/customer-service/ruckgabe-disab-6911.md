# Rückgabe: disableCustomer

Rückgabe disableCustomerResponse

| Attribut | Bedeutung | Typ und Wertebereich |
| --- | --- | --- |
| success | Es handelt sich um einen asynchronen Deaktivierungsvorgang. Wenn TRUE als ein valider Request übergeben wurde, dann startet der Deaktivierungsprozess. Dieser Prozess kann eine geraume Zeit dauern, bis alle Deaktivierungen vorgenommen worden sind.  Wird FALSE zurückgegeben ist die Kundennummer ungültig und die Deaktivierung gescheitert. | Boolean |
