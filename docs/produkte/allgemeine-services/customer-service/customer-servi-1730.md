# Customer Service Schnittstellenoperationen

Die DAT stellt ihren Kunden einen Webservice unter der folgenden Adresse zur Verfügung.
Die DAT generiert damit ihren externen Kunden einen eigenen Zugang.

```
http://www.dat.de/AuthorizationManager/soap/CustomerService
```

Die technische Metabeschreibung des Services als WSDL (Web Services Description Language)
kann über die folgende Adresse abgefragt werden:

```
http://www.dat.de/AuthorizationManager/soap/CustomerService?wsdl
```

Für alle SOAP-Methoden ist eine Authentifizierung erforderlich. Die Zugangsdaten (engl.
Credentials) werden jeweils über den HTTP Header übermittelt.
