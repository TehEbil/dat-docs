# Schnittstellenoperationen

Die spezifischen Schnittstellen-Funktionen der SilverDAT 3 valuateFinance zur Bewertung bzw. zum Verwalten der Vorgänge und ihre Beschreibungen rufen Sie über
folgende URL's auf:

WSDL

<https://www.datgroup.com/FinanceLine/soap/Evaluation?wsdl>

<https://www.datgroup.com/FinanceLine/soap/DossierN?wsdl>

<https://www.datgroup.com/FinanceLine/soap/BatchValuation?wsdl>

Serviceaufruf

https://www.datgroup.com/FinanceLine/soap/Evaluation

https://www.datgroup.com/FinanceLine/soap/DossierN

https://www.datgroup.com/FinanceLine/soap/BatchValuation
