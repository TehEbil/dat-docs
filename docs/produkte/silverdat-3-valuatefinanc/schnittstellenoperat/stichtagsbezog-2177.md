# Stichtagsbezogene Bewertung durchführen (Historische Bewertung)

Die Funktion getVehicleTargetDateEvaluationHistory bietet die Möglichkeit, eine stichtagsbezogene oder auch "Historische" Bewertung durchzuführen.

Rückgabe

Durch die Übergabe von condition und seiner Unterelemente kann eine Wertkorrektur durchgeführt werden. Im Erfolgsfall
wird das Ergebnis als VXS zurückgeliefert, ansonsten wird eine Fehlermeldung zurückgegeben. Die Beschreibung
der Response-Elemente finden Sie im Kapitel VXS.
