# Funktionsumfang Vorgangsverwaltung

Mit Hilfe der Dossier-Funktionen verwalten Sie Ihre Vorgänge über die Soap Schnittstelle.

Hinweis: Das Produkt SilverDAT 3 valuateFinance benötigt keine Produktvariante. Bei SilverDAT 3 valuateExpert und SilverDAT 3 valuateExpertPlusPartner muss eine Produktvariante angegeben werden.

Übersicht der Funktionen und deren Verfügbarkeit

| Funktion | Beschreibung | SilverDAT 3 valuateFinance | SilverDAT 3 valuateExpert SilverDAT3 valuateExpertPlusPartner |
| --- | --- | --- | --- |
| [createDossierN](#showid/14999 "Erstellen eines neuen Vorgangs") | Erstellen von neuen Vorgängen. | X | X |
| [getDossierN](#showid/15007 "Holen eines Vorgangs") | Abrufen von bestehenden Vorgängen. | X | X |
| [changeDossierN](#showid/14989 "Aendern eines Vorgangs") | Ändern von Vorgängen. | X | X |
| [updateDossierN](#showid/14991 "Aktualisieren eines Vorgangs ohne Wertänderung") | Aktualisieren der Bewertung eines Vorgangs. | X | X |
| [exportDossierN](#showid/12657 "Exportieren von Dokumenten (SilverDAT 3 valuateExpert, valuateExpertPlusPartner)") | Erstellen von druckfähigen Dokumenten mit Urkundennummer und BAFA Gutachten |  | X |
| [exportDossierN](#showid/1506 "Exportieren von Dokumenten (SilverDAT 3 valuateFinance)") | Ausdrucke erstellen | X |  |
| [resetDossier2DefaultN](#showid/15072 "Zurücksetzen eines Vorgangs auf die DAT-Vorgaben") | Zurücksetzen eines Vorgang auf die DAT Standardeinstellungen. | X | X |
| [resetDossierTechDataN](#showid/25260 "Zurücksetzen von technischen Daten eines Vorgangs auf die DAT-Vorgaben") | Zurücksetzen von technisches Daten eines Vorgangs auf die DAT Standardeinstellung | X | X |
| [deleteDossierN](#showid/15009 "Löschen eines Vorgangs") | Löschen von bestehenden Vorgängen. | X | X |
| [searchDossierListN](#showid/15068 "Suchen von Vorgängen") | Auffinden von bestehenden Vorgängen. | X | X |

Die spezifischen Schnittstellen-Funktionen derSilverDAT 3 valuateFinance, SilverDAT 3 valuateExpert und SilverDAT 3 valuateExpertPlusPartner zum Verwalten der Vorgänge und ihre Beschreibungen rufen Sie über folgende URL's
