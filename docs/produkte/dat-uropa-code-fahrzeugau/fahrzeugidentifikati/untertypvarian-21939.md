# Untertypvariante nicht gefunden

Beispiel Test-VIN VSSDATTESTSTUB002

In diesem Beispiel konnte die Untertypvariante nicht gefunden werden (SubModelVariant = 0), damit ist der DAT €uropa-Code® nicht vollständig. Dieses wird mit der Fehlermeldung "ECode not found" angezeigt. Nachfolgend viele weitere Daten vom Hersteller, wie Herstellercodes und
Herstellertexte. Weiter unten zu sehen, sind die DAT-Schlüssel die den Herstellerdaten
zugeordnet werden konnten.
