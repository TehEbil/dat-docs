# Voraussetzungen

Es wird zwingend ein Zugang zur DAT €uropa-Code® Fahrzeugauswahl benötigt. Dieser
besteht aus gültigen Authentifizierungsdaten, die sich aus DAT Kundennummer, Benutzername
und Passwort zusammensetzt.

Weiterhin muss die Domain [https://www.dat.de](https://www.datgroup.com "https://www.datgroup.com") im Browser erreichbar sein.

Für den produktiven Einsatz der Schnittstelle wird im Hinblick auf die Datensicherheit
die Verwendung des HTTPS-Protokolls erforderlich.
