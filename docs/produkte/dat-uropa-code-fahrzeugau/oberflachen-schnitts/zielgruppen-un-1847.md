# Zielgruppen und notwendige Vorkenntnisse

Zielgruppen dieses Kapitels sind einerseits Entscheidungsvorbereiter, die die Machbarkeit
angedachter Implementierungen für einen oder mehrere Prozesse zur Fahrzeugidentifikation
mit Benutzerinteraktion prüfen und andererseits Entwickler, die eine Fahrzeugidentifikation
durch Integration der DAT €uropa-Code® Fahrzeugauswahl in ihrer Anwendung umsetzen.

Die notwendigen Vorkenntnisse umfassen ein grundlegendes Verständnis für die Integration
verschiedener Webanwendungen. Dies schließt die Kenntnis des HTTP-Protokolls sowie
der grundlegenden technischen Möglichkeiten innerhalb eines Browsers mittels HTML,
CSS und JavaScript ein. Entwickler sollten darüber hinaus genaue Kenntnisse in der
dynamischen Manipulation des Document Object Models (DOM) via JavaScript besitzen
sowie mit der Aufbereitung und Auswertung von XML-Daten vertraut sein.
