# kleiner Exkurs: DAT €uropa-Code® und Marktindex

Der DAT €uropa-Code® setzt sich aus einem 15-stelligen Basiscode zusammen. Ergänzt
wird der DAT €uropa-Code® noch durch den 9-stelligen Marktindex, der damit ein Fahrzeug
eindeutig identifizieren kann:

Der 15-stellige Basiscode des DAT €uropa-Code® setzt sich aus folgenden Komponenten
zusammen (Auflistung in der Reihenfolge der Darstellung):

| Schlüssel | Beispiel |
| --- | --- |
| Fahrzeugart | PKW, SUV, Kleintransporter |
| Hersteller | Mercedes-Benz |
| Haupttyp | BM 211 E-Klasse Lim. ab 01/2002 |
| Untertyp | E 220 CDI |
| Typvariante | Motor 2,2l 110 KW, 4-türig, Automatik, 5-Stufen, Elegance |

Das 9-stellige Zusatzelement1, oder "Marktindex", wie er heute genannt wird, setzt
sich aus zwei Komponenten zusammen:

| Schlüssel | Beispiel |
| --- | --- |
| (DAT) Container | DE, Lim4 220 CDI Euro-Norm 4, Elegance DPF, 2003-2006 |
| Bauzeit | 4330 |
