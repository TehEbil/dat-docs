# Mögliche Fehlercodes getEngineOptions

| Rückgabecode | Fehlertext | Beschreibung |
| --- | --- | --- |
| wrongValueCode | wrong equipment type | Ausstattungstyp falsch. |
| valueNotFound | No result for the selected options. | Mit der übergebenen Ausstattung kann kein eindeutiges oder gar kein Ergebnis erzielt werden. |
| valueNotFound | no results found. Wrong <code>. | Bitte prüfen Sie den <Schlüssel>. |
| valueNotFound | [type] not found for construction time range. | Die übergebenen Bauzeiten führen zu keinem Ergebnis. Wobei [type] der abgefragte Typ ist, z.B. Manufacturers, Base Models. |
| valueNotFound | not results found for construction time range. | Die übergebenen Bauzeiten führen zu keinem Ergebnis. Es wurden Ausstattungen zu einem DAT €uropa-Code® abgefragt. |
