# Jahr und Monat der stichtagsbezogenen Bewertung erfragen

Mit der Funktion getYearMonthsForVehicleTargetDateEvaluationHistory können Sie die möglichen Jahr/Monatskombinationen für die stichtagsbezogene Bewertung
für ein bestimmtes Datenland abfragen.
