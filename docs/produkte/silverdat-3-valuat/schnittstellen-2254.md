# Schnittstellenoperationen

Die Produkte SilverDAT 3 valuateExpert/PlusPartner stellen Fremdsystemen verschiedene SOAP Webservices zur Verfügung. Folgende Webservices
werden in diesem Kapitel näher beschrieben:

1. [Authentifizierung](authentifizier-2256.md)
2. [Fahrzeugidentifikation](fahrzeugidenti-2259.md)
3. [Fahrzeugauswahl](fahrzeugauswah-2261.md)
4. [Konvertierungsoperationen](konvertierungs-2263.md)
5. [Fahrzeugbilder](fahrzeugbilder-2265.md)
6. Verwaltung von Vorgängen, Bewertungen und Ausdrucken
7. Expert-Funktionen

Bitte beachten Sie, dass zum Ansprechen der Schnittstellenfunktionen der SilverDAT 3 valuateExpert/valuateExpertPlusPartner zusätzlich zu den Anmeldedaten die Produktvariante im HTTPS-Header gesetzt werden
muss.

Produktvarianten

| Produkt | Produktvariante | Pflicht |
| --- | --- | --- |
| SilverDAT 3 valuateExpert | valuateNG.expert | X |
| SilverDAT 3 valuateExpertPlusPartner | valuateNG.expertPlusPartner | X |

Die spezifischen Schnittstellen-Funktionen der SilverDAT 3 valuateExpert/valuateExpertPlusPartner zur Bewertung und zum Verwalten der Vorgänge und ihre Beschreibungen rufen Sie über
folgende URL's auf:

WSDL

<https://www.datgroup.com/valuateNG/soap/Dossier1N?wsdl>  
<https://www.datgroup.com/valuateNG/soap/ExpertServiceN?wsdl>  
<https://www.datgroup.com/valuateNG/soap/VehicleUserPropertiesService?wsdl>  
<https://www.datgroup.com/valuateNG/soap/RentalPricesService?wsdl>

Serviceaufruf

https://www.datgroup.com/valuateNG/soap/Dossier1N  
https://www.datgroup.com/valuateNG/soap/ExpertServiceN  
https://www.datgroup.com/valuateNG/soap/VehicleUserPropertiesService  
https://www.datgroup.com/valuateNG/soap/RentalPricesService
