# Prämissen der DAT für neue Anwendungen

Folgende oberste Prämissen gelten für alle neuen DAT-Anwendungen:

Zuerst ist ein Fahrzeug zu identifizieren, danach kann das identifizierte Fahrzeug
bewertet oder eine Restwertprognose vorgenommen werden.

1. Ein identifiziertes Fahrzeug hat einen vollständigen DAT €uropa-Code®, bei der Bewertung besteht dieser aus den 15stelligen DAT €uropa-Code® inklusive dem 9stelligen Marktindex.
2. Nur ein vollständiger DAT €uropa-Code® kann Daten ins DAT System bringen.
3. Schnittstellenfunktionen, die Aktionen ohne vollständigen DAT €uropa-Code® ermöglichen, speichern Daten NICHT ins System.
