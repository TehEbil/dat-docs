# Verwalten der Fahrzeugbeschreibung

Mithilfe der nachfolgend beschriebenen Funktionen verwalten Sie die Fahrzeugbeschreibung
eines Vorgangs. Diese Funktionen finden Sie nur unter dem Service VehicleUserPropertiesService für die folgende Anwendung verfügbar: SilverDAT 3 valuateExpert/valuateExpertPlusPartner.

Übersicht der Funktionen

| Funktion | Beschreibung |
| --- | --- |
| setVehicleColorUpholstery | Fahrzeugbeschreibung anlegen |
| getVehicleColorUpholstery | Fahrzeugbeschreibung holen |

WSDL

https://www.datgroup.com/valuateNG/soap/VehicleUserPropertiesService?wsdl

Serviceaufruf

https://www.datgroup.com/valuateNG/soap/VehicleUserPropertiesService
