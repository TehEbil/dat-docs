# Funktionsumfang

Mithilfe der nachfolgend beschriebenen Funktionen verwalten Sie die Grundüberprüfung,
Leasingrückgabe, Zustandsberichte, Bewertungsgutachten, Schätzungsurkunden, Kontakte,
Auftragsdaten sowie auch Zustandsdetails in der SilverDAT 3 valuateExpert/valuateExpertPlusPartner. Diese Funktionen finden Sie nur unter dem Service ExpertServiceN.
