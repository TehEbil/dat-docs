# Schema anlegen für den Zustandsbericht

###### Schema anlegen für den Zustandsbericht Mit der Funktion setProductCheckDetailsList legen Sie ein Schema zu einem bestehenden Vorgang an. Wenn Sie für Usage den Wert CR angegeben haben, werden Prüfpunkte im Produkt ConditionReport angelegt. Die Kindelemente von ConditionDetails sind nur für das Produkt Zustandsbericht verfügbar.
