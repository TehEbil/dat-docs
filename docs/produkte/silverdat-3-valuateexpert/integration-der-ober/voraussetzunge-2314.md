# Voraussetzungen SilverDAT 3 valuateExpert Oberflächenintegration

Um die Oberfläche der SilverDAT 3 valuateExpert/PlusPartner in ihre Anwendung zu integrieren, benötigen Sie zwingend ein Zugang zur SilverDAT 3 valuateExpert/PlusPartner. Zur Authentifizierung muss der [DAT-AuthorizationToken](../../../allgemein/dat-developers-guide/authentifizierung-de/json-web-token-14087.md) verwendet werden. Außerdem muss die Domain [https://www.datgroup.com](https://www.datgroup.com "https://www.datgroup.com") im Browser erreichbar sein. Ihre Anwendung muss bei diesem Anwendungsfall die Fähigkeit
besitzen als Web-Server zu agieren, eine weitere Web-Anwendung in Form einer HTML-Seite
einzubinden und mit ihr zu kommunizieren. Für eine gute Darstellung sollte mindestens
eine Auflösung von 1280 x 800 Pixel gegeben sein.

Bitte beachten Sie, dass bei der Oberflächen-Integration unserer Produkte mittels
iFrame alle URLs die sich auf Produkte oder Ressourcen der DAT beziehen, das HTTPS-Protokoll verwenden müssen.
