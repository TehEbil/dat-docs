# getEquipmentGroups

Beschreibung

Ermittelt die für das entsprechende Land verfügbaren Ausstattungsvarianten.

Parameter

| Name | Datentyp | Mögliche Werte | Beschreibung | Pflichtfeld |
| --- | --- | --- | --- | --- |
| - | - | - | - | - |

Rückgabe

Zurückgegeben wird eine Liste aller verfügbaren Ausstattungsvarianten für das jeweilige
Land in der jeweiligen Landessprache.
