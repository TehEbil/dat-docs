# Rückgabewert

Im Falle einer gültigen Übergabe von Daten erfolgt ein valider Response ohne Daten und Meldungen. Bei einer ungültigen Übergabe wird eine Fehlermeldung ausgegeben.

Bitte sehen Sie sich hierzu auch den [Beispiel-Response zu createContract](beispiel-respo-2118.md) an.
