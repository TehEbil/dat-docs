# Nutzungsvoraussetzungen

Voraussetzungen zur Verwendung dieser Webservices:

1. Sie haben eine gültige Lizenz für dieses Produkt. Weitere Informationen unter [sales-support@dat.de](mailto:sales-support@dat.de "Mail an DAT Sales Support") oder [vertrieb@dat.de](mailto:vertrieb@dat.de "Mail an DAT Vertrieb").
2. Sie haben die Schnittstellenrechte für dieses Produkt.   
   Bitte informieren Sie unseren Vertrieb darüber, dass Sie auch die zugehörigen Schnittstellen
   nutzen möchten.

Sie haben sich als Schnittstellenpartner registrieren lassen.   
Als Schnittstellenpartner erhalten Sie regelmäßige Informationen über wichtige Änderungen
in den Schnittstellen. Außerdem erhalten Sie die Möglichkeit, Ihre notwendigen Schnittstellen-Dokumentationen
zu aktualisieren. Beide Informationen benötigen Sie, um zukünftige Fehler in Ihren
Anwendungen durch die Benutzen unserer Schnittstellen zu vermeiden. Bitte senden Sie
eine E-Mail unter Angabe folgender Informationen an interfaces@dat.eu:

- Kundennummer
- Name der Kontaktperson in Ihrem Unternehmen
- E-Mail-Adresse der Kontaktperson

1. Sie verwenden eine gültige Fassung der Schnittstellen-Dokumentation.  
   Diese erhalten Sie regelmäßig, nachdem Sie sich als Schnittstellenpartner registriert
   haben.
2. Sie verwenden eine gültige URL zum Aufruf des Webservices.   
   Sie finden den Link zur WSDL im Kapitel [WSDL-URLs für das Produkt autoglas Plus /SilverDAT calculateGlass](../grundlagenwissen/wsdl-urls-fur-2082.md).
