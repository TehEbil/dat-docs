# Rückgabewert getNewContracts

Die Rückgabe der Daten erfolgt als VXS. Sind keine nicht-abgerufenen, versendeten Kalkulationen vorhanden, wird ein leeres
VXS zurückgegeben. Es erfolgt eine Rückgabe von Komplettdaten.
