# SparePartsSortingCriterions

Die Ergebnisliste kann mit den folgenden Parametern sortiert werden:

| Attribut | Typ | Beschreibung |
| --- | --- | --- |
| [baseModel](orderinfo-6718.md) | complexType, optional, [OrderInfo](orderinfo-6718.md) | Die Ergebnisliste wird nach Haupttyp sortiert (auf- oder absteigend) |
| [subModel](orderinfo-6718.md) | complexType, optional, [OrderInfo](orderinfo-6718.md) | Die Ergebnisliste wird nach Untertyp sortiert (auf- oder absteigend) |
| [constructionTime](orderinfo-6718.md) | complexType, optional, [OrderInfo](orderinfo-6718.md) | Die Ergebnisliste wird nach Bauzeit sortiert (auf- oder absteigend) |
