# paintingColorDescription

Definiert die Bezeichnung, Art und Eigenschaften des Lacks.

| Attribut | Bedeutung | Typ und Wertebereich |
| --- | --- | --- |
| code |  | String |
| datCode | DAT-Farbe/-Farbnummer. Mögliche Werte: 0 => Weiß; 1 => Gelb; 2 => Orange; 3 => Rot; 4 => Lila; 5 => Blau;  6 => Grün; 7 => Grau/Silber; 8 => Braun;  9 => Schwarz | Integer |
| expression | Zusatz für DAT-Farbe/Farbnummer. Mögliche Werte: 1 => hell; 0 => ohne; 2 => dunkel | Integer |
| name | Farbbezeichnung; z.B. Gelb, dunkel, metallic | String |
| priceCategory |  | String |
| type | Möglicher Wert: 1 => metallic | Integer |
