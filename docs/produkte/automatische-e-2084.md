# automatische Erzeugung eines Vorgangs mit createContract

Ein Glas-Vorgang aus einer Fremdapplikation soll einen neuen Vorgang in autoglas Plus erzeugen. Dazu müssen die erforderlichen Daten als VXS übergeben werden.
