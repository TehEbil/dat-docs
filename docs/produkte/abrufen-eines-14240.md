# Abrufen eines Adressbuchs

Liefert anhand der Adressbuch ID die spezifische Kontaktinformation einer Person aus
dem Adressbuch

Parameter

| Name | Datentyp | Mögliche Werte | Beschreibung | Pflichtfeld |
| --- | --- | --- | --- | --- |
| Id | Long |  | Adressbuch ID zum Filtern des Kontaktperson Adressbuchs | X |

Rückgabe

Liefert ein Adressbuch das die Kontaktperson mit Fahrgestellnummer und Kennzeichen
beinhaltet zurück.
