# Mögliche Fehlercodes getInsurances

| Fehlercode | Fehlertext | Beschreibung |
| --- | --- | --- |
| Server.CountryError | wrong country value | Tritt auf, wenn beim Parameter "country" ein ungültiger / kein Wert eingetragen wird. |
| Server.LocaleError | wrong country indicator value | Bitte prüfen Sie den Parameter "locale", er enthält ungültige Attributwerte. |
| Server.CountryError | wrong language value | Tritt auf, wenn beim Parameter "language" ein ungültiger / kein Wert eingetragen wird. |
