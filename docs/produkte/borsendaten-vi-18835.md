# Börsendaten via webScan REST-API abrufen

Die nachfolgend beschriebenen Funktionen bieten die Möglichkeit einen webScan für
ein bereits angelegtes, sowie für ein identifiziertes Fahrzeug mittels DATECode, über
die REST-Schnittstelle durchzuführen.
