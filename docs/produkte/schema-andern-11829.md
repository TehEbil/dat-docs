# Schema ändern

###### Schema ändern Mit der Funktion changeProductCheckDetailsList legen Sie ein Schema zu einem bestehenden Vorgang an. Der Parameter Usage ist nicht änderbar. Wenn sie Usage ändern wollen, müssen Sie mit setProductCheckDetailsList ein neues Schema anlegen.
