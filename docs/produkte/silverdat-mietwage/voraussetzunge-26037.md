# Voraussetzungen SilverDAT 3 Mietwagenspiegel Oberflächenintegration

Um die Oberfläche des Produkts SilverDAT 3 Mietwagenspiegel in ihre Anwendung zu integrieren, benötigen Sie zwingend einen Zugang zum SilverDAT 3 Mietwagenspiegel. Zur Authentifizierung muss der [DAT-AuthorizationToken](../../allgemein/dat-developers-gui/json-web-token-14087.md) verwendet werden. Außerdem muss die Domain [https://www.datgroup.com](https://www.datgroup.com "https://www.datgroup.com") im Browser erreichbar sein. Ihre Anwendung muss bei diesem Anwendungsfall die Fähigkeiten
besitzen als Web-Server zu agieren, eine weitere Web-Anwendung in Form einer HTML-Seite
einzubinden und mit ihr zu kommunizieren. Für eine gute Darstellung sollte mindestens
eine Auflösung von 1280 x 800 Pixel gegeben sein.

Bitte beachten Sie, dass bei der Oberflächen-Integration unserer Produkte mittels
iFrame alle URLs, die sich auf Produkte oder Ressourcen der DAT beziehen, das HTTPS-Protokoll verwenden müssen.
