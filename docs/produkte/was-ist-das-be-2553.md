# Was ist das Besondere an SilverDAT 3 PRO?

SilverDAT 3 PRO ist ein leistungsfähiges und umfassendes Werkzeug für Automobilbetriebe. Es basiert
auf den Erkenntnissen von über 25 Jahren Produktentwicklung bei SilverDAT II und über
85 Jahren DAT Know-how. SilverDAT 3 PRO dient der professionellen Identifikation von Fahrzeugen, der Wertermittlung z.B.
bei der Inzahlungnahme, der Kalkulation von Reparaturkosten und es unterstützt die
Anwender beim Schadenmanagement.

Nahezu alle enthaltenen Funktionen können auch automatisiert über die Schnittstellen
angesprochen werden.
