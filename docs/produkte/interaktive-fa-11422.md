# interaktive Fahrzeugauswahl

Analog zur Fahrzeugauswahl über den klassischen Suchbaum steht auch für die interaktive
Suche nach Fahrzeugen eine Schnittstellenfunktion zur Verfügung. Wie in unseren Web
Anwendungen auch, ist diese eingeschränkt auf die Suche nach Fahrzeugen zur Gebrauchtfahrzeugbewertung.
Fahrzeuge älter als 20 Jahre können damit leider nicht ermittelt werden.
