# Abrufen der Status zu einem Netzwerk

Diese Funktion listet alle Status aus einem Netzwerk auf.

Parameter

| Name | Datentyp | Mögliche Werte | Beschreibung | Pflichtfeld |
| --- | --- | --- | --- | --- |
| networkType | String | DAT | Konstante eines bestimmten myclaim Netzes | X |

Rückgabe

Liste aller verfügbaren Netzwerkstatus
