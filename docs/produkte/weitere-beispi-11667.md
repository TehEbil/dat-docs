# Weitere Beispiele (mit speziellem Einstieg)

Es besteht die Möglichkeit eines Einstiegs in die interaktive Fahrzeugauswahl mit
einer Liste von einem oder mehreren DAT €uropa-Codes®. Diese können aus einer vorgelagerten Abfrage z.B. getVehicleIdentificationByKba oder auch über getVehicleIdentificationByTextSearch als unvollständige DAT €uropa-Codes® ermittelt werden.
