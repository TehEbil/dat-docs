# Mögliche Fehlercodes calculateContract()

| --- | --- | --- |
| Server.methodHasNoRequest | methodHasNoRequest | Sie haben keinen Request übergeben. |
| dat:Server.authorizationFailed | Server.authorizationFailed | Das Token ist abgelaufen oder ungültig. Erstellen Sie ein neues Token oder überprüfen Sie Ihre Anmeldedaten. |
| S:Server | Unable to find contract with id = 111111 | Die contractID ist ein Pflichtparameter. Bitte geben Sie die contractID ein. |
| dat:Server.valueNotFound | contractID | Die contractID ist ein Pflichtparameter. Bitte übergen die contractID. |
| Server.valueNotFound | Contract [contractId] is not complete for calculation. Necessary entire datECode | Bitte überprüfen Sie die Vorgangsdaten. |
| Server.calculationError | Error with calculation the result. | Bestimmte Gründe sind möglich. Bitte kontaktieren Sie [interfaces@dat.eu](mailto:interfaces@dat.eu "Mail DAT interfaces"). |
