# ExportProduct

Parentnode enthält Informationen zur Druckvorlage

| Attribut | Typ | Mögliche Werte | Beschreibung |
| --- | --- | --- | --- |
| displayName | String, optional |  | Name der Druckvorlage |
| filename | String, optional |  | Dateiname der Druckvorlage |
| identifier | Long, optional |  | Identifikationsnummer der Druckvorlage |
| origin | String, optional | "Eigene Vorlage", "DAT-Standard" | Ursprung der Druckvorlage |
| printProductName | Integer | "CALC\_ESTIMATE", "CALC\_INVOICE", "CALC\_PARTS\_SPO", "INSPECTION\_REPORT"  "CALC\_BLANKET"  "CALC\_ESTIMATE\_CH"  "CALC\_INVOICE\_CH"  "CALC\_ESTIMATE\_ITALY"  "CALC\_INVOICE\_ITALY"  "CALC\_PARTS\_ITALY"  "CALC\_IMAGES\_ITALY" | Druckprodukt |
