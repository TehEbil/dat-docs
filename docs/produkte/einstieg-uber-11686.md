# Einstieg über einen unvollständigen DAT €uropa Code

Der verwendete unvollständige DAT €uropa-Codes® wurde aus der Freitextsuche ermittelt. Die daraus ermittelten Schlüssel (im Beispiel
Fahrzeugart, Hersteller, Haupttyp, Untertyp) werden zum unvollständigen DAT €uropa-Codes® zusammengesetzt und an die interaktive Fahrzeugauswahl übergeben. Der Beispielaufruf
liefert 24 konkrete Fahrzeuge. Bei Bedarf kann über die Filteroptionen weiter eingeschränkt
werden.

- Suche nach Fahrzeugen im Markt Deutschland, Ausgabe der Daten in deutsch  
  ("locale": ...).
- Übergabe eines unvollständigen DAT €uropa-Code® zur weiteren Filterung, bis ein Fahrzeug gefunden wurde.  
  ( "datECodes": [...] )
