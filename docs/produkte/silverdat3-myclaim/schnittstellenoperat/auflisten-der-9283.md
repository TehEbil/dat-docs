# Auflisten der vorhandenen Netzwerktypen

Diese Funktion listet die Konstanten aller verfügbaren Netzwerktypen einer Kundenummer
auf.

Parameter

| Name | Datentyp | Mögliche Werte | Beschreibung | Pflichtfeld |
| --- | --- | --- | --- | --- |
| templateTypes | String | vro\_calculation, bundle\_valuation | Auftragstypen (Schadenkalkulation, Bewertung) | X |

Rückgabe

Konstanten aller verfügbaren Netzwerktypen
