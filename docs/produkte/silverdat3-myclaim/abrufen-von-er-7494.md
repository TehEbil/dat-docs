# Abrufen von Ersatzteilen aus einer Kalkulation

Mit dieser Funktion können Ersatzteile innerhalb einer Kalkulation aufgelistet bzw.
exportiert werden.

Parameter

| Name | Datentyp | Mögliche Werte | Beschreibung | Pflichtfeld |
| --- | --- | --- | --- | --- |
| contractId | Long |  | ID des Auftrags | X |

Rückgabe

Liste mit den Ersatzteilen der Kalkulation
