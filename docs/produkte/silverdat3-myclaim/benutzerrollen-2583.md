# Benutzerrollen

Jeder Benutzer in myClaim ist einer bestimmten Rolle zugeordnet. Auch kann ein Benutzer in seinem Netz mit verschiedenen
Partnern/Akteuren interagieren.

Parameter

| Wert im Parameter statusType | Beschreibung |
| --- | --- |
| INSURANCE | Versicherungsgesellschaft |
| MANUFACTURER | Hersteller |
| EXPERT | Sachverständiger |
| REPAIRER | Werkstatt |
| FLEET | Flotte |
| GUEST | Gast |
| LAWYER | Anwalt |
| OTHER\_PARTNER | Partner |
| SERVICE\_PROVIDER | Dienstleister |
