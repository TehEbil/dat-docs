# Datenabruf in länderspezifischer Sprache

Um einen Datenabruf in einer bestimmten Sprache durchzuführen, gibt es die Möglichkeit
durch das separate HTTP-Header Element namens Language, den Country Code für die gewünschte Sprache im Request mitzugeben (siehe Screenshot).

Nicht alle Funktionen und Datenelemente können länderspezifisch abgerufen werden.
