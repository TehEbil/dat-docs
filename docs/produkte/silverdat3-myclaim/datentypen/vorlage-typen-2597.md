# Vorlage Typen (templateTypes)

In myClaim unterscheidet man neben den Auftragstypen noch die Vorlagentypen. Ein Auftrag
ist immer ein Teil eines Vorlagentyps. Ein Vorlagen Typ ist die Art der Oberfläche
bzw. des Arbeitsbereichs innerhalb von myClaim.

Alle möglichen Vorlagetypen sind hier unterhalb aufgelistet. Dabei ist der gängigste
Typ einer Schadenskalkulation "vro\_calculation" für Karosserie und Lackschäden und für reine Glaskalkulationen ist es der Typ "calculation".

|  |  |
| --- | --- |
| Wert im Parameter templateTypes | Beschreibung |
| custom | Zusätzliche kundenspezifische Daten |
| offlineOpen | Offline: Kalkulationsauftrag |
| onlineOpen | Online: Kalkulationsauftrag |
| offlineUpload | Offline: Kalkulation zum bestehenden Auftrag hochladen |
| glassInvoiceRates | Online: Glas Verrechnungssatz |
| glassAdminInvoiceRates | Online: Glas Admin Verrechnungssatz |
| calculation | Online: Glas Berechnung in einem Template |
| contactPerson | Online: Ansprechpartner |
| vro\_calculation | Online: FI Kalkulation in einem Template |
| test | Test Konditionen |
| companyData | Online: Unternehmensinfo |
| manualPosition | Online: Manuelle Position |
| operationSupply | Online: Hilfs- und Betriebsstoffe |
| wagesSettings | Online: Werkstattdaten |
| userSettings | Online: Benutzereinstellungen |
| vro\_fi\_calculation | FIo Post Schnistelle |
| GDVRelease2000 | Online: GDV Release 2000 |
| GDVRelease2003 | Online: GDV Release 2003 |
| legend | Legende: Teileauswahl |
| vro\_domus\_calculation | Online: FI-Domus Kalkulation in einem Template |
