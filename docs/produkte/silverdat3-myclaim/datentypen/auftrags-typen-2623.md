# Auftrags Typen (contractTypes)

In myclaim können verschiedene Typen eines Auftrags bzw. einer Akte angelegt werden.
Folgende Auftragsarten sind in myclaim bisher möglich:

Hinweis

Die gängigsten Auftrgafstypen (contractTypes) sind vro\_calculation für Fahrzeuginstandsetzungen und bundle\_valuation für Bewertungen

|  |  |
| --- | --- |
| Name | Datentyp |
| vro\_calculation | Fahrzeuginstandsetzungs Kalkulation in einer Auftragsvorlage. |
| bundle\_valuation | Für Bewertungsvorgänge |
| glass | Glas Kalkulation in einer Auftragsvorlage (Template). |
| general | Generelle Auftragsanlage über eine Auftragsvorlage oder aus der Übertragung eines Auftrags aus SDII. |
| ania | Spezielles Kalkulationsformat für den italienischen Markt. |
| vsr\_order | Karosserie und Lackkalkulation direkt in einer Auftragsvorlage (Template) mit der Verwendung des FI Webservices (VehicleRepairService) |
| vro\_fi\_calculation | Fahrzeuginstandsetzungs Kalkulation in einer Auftragsvorlage mit der Verwendung des FI Webservices (VehicleRepairService). |
