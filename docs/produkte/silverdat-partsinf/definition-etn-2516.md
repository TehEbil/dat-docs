# Definition ETN und DVN

In dieser Schnittstellen-Dokumentation wird von ETN und DVN gesprochen, wenn die Teilekennung gemeint ist. In der reinen Definition unterscheiden
sich ETN und DVN folgendermassen:

ETN

Die ETN ist die Ersatzteilnummer des Herstellers. Die ETN wird in der Silver DAT-Welt auch als Teile-Nummer bezeichnet. Sie ist alphanumerisch (kann alphanumerisch
sein) und Leerzeichen enthalten.

Hier ist sie ist die als externe Teile-Nr. bezeichnete Teilekennung. Ein Beispiel
wäre: 1T09073A FKZ

DVN

Die DVN ist eine DAT-interne Datenverarbeitungsnummer. Sie ist Bestandteil eines Nummernsystems,
der DVN-Tabelle, das HST-übergreifend Fahrzeugteile mit einem Code belegt. Die DVN wird durch eine 5-stellige Nummer dargestellt. Man benötigt sie um Fahrzeugteile in
der Datenbank anzulegen. Für jedes Teil, das in der Datenbank verwendet wird, ist
eine DVN festgelegt. Für gleiche Teile ist auch die gleiche DVN zu verwenden <z.B. 43711 Kotflügel v.l.>. In Verbindung mit einem Reparaturcode  lässt sich ein Reparaturvorgang codieren. Die Datenverarbeitungsnummer wird gelegentlich
auch als DAT-Vorgangs- oder Verarbeitungsnummer bezeichnet.

Hier ist sie die als interne Teile-Nr. bezeichnete Teilekennung. Ein Beispiel wäre:
41210
