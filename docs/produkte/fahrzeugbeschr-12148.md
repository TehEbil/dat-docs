# Fahrzeugbeschreibung holen

Mit der Funktion getVehicleColorUpholstery holen Sie eine Fahrzeugbeschreibung zu einem bestehenden Vorgang. Diese Methode ist
für folgende Anwendungen verfügbar: SilverDAT 3 valuateExpert/valuateExpertPlusPartner.

Parameter getVehicleColorUpholstery

| Name | Datentyp | Beschreibung | Schreibweise | Pflicht |
| --- | --- | --- | --- | --- |
| DossierId | Integer | Eindeutige Vorgangserkennung | numerisch | x |
