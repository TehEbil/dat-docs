# Einstieg über mehrere DAT €uropa Codes

Die verwendeten DAT €uropa Codes® stammen von einer KBA-Abfrage und führen schon mit dem ersten Aufruf auf 6 konkrete
Fahrzeuge. Bei Bedarf kann über die Filteroptionen weiter eingeschränkt werden.

- Suche nach Fahrzeugen im Markt Deutschland, Ausgabe der Daten in deutsch  
  ("locale": ...).
- Übergabe von mehreren DAT €uropa Codes® zur weiteren Filterung, bis ein Fahrzeug gefunden wurde.  
  ( "datECodes": [...] )
