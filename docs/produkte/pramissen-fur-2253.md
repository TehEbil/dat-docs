# Prämissen für SilverDAT 3 valuateExpert

Folgende oberste Prämisse gilt für SilverDAT 3 valuateExpert/PlusPartner:   
Zuerst ist ein Fahrzeug zu identifizieren, danach kann das identifizierte Fahrzeug
bewertet werden.

Ein identifiziertes Fahrzeug hat einen vollständigen DAT €uropa-Code®, bei der Bewertung besteht dieser aus den 15stelligen DAT €uropa-Code® inklusive dem 9stelligen Marktindex.
