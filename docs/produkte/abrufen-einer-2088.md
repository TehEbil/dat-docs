# Abrufen einer Liste von Vorgangsnummern mit getNewConctractNumbers

Mit dieser Funktion wird eine Liste von noch nicht versendeten Kalkulationen abgerufen.

Es werden nur Vorgangsnummern angezeigt, die vorher noch nicht mit getContract() oder getNewContracts() abgerufen wurden.
