# Schnittstellenoperationen

Das Produkt SilverDAT Mietwagenspiegel stellt Fremdsystemen zwei SOAP Webservices zur Verfügung. Diese Webservices werden
in diesem Kapitel näher beschrieben:

1. [Authentifizierung](#showid/7680 "Authentifizierung")
2. [Mietwagenspiegel-Funktionen](#showid/7685 "Mietwagenspiegel-Funktionen")

Die spezifischen Schnittstellen-Funktionen der SilverDAT Mietwagenspiegel rufen Sie über die folgende URL's auf:

WSDL

<https://www.datgroup.com/rentalPrices/soap/Authentication?wsdl>  
<https://www.datgroup.com/rentalPrices/soap/RentalPricesService?wsdl>

Serviceaufruf

https://www.datgroup.com/rentalPrices/soap/Authentication  
https://www.datgroup.com/rentalPrices/soap/RentalPricesService
