# Adressbuchverwaltung

Folgende Funktionen bieten Drittanbietern die Möglichkeiten Adressinformationen nach
myclaim zu übertragen die dem myclaim Benutzer anschließend im Template als Auswahl
zur Verfügung stehen.

listAddressBook

Listet alle Adressbücher entsprechend der Filterkriterien

getAddressBook

Liefert Informationen über das Adressbuch

saveAddressBook

Wird verwendet, um ein neues Adressbuch zu erstellen oder ein vorhandenes Adressbuch
zu aktualisieren

removeAddressBook

Entfernt ein Adressbuch über dessen ID
