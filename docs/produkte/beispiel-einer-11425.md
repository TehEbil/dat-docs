# Beispiel einer Fahrzeugauswahl

Beispielaufrufe der interaktiven Fahrzeugauswahl zur Ermittlung eines SUV. Am Ende
der Schnittstellenaufrufe wird ein einzelnes Fahrzeug zur Weiterverarbeitung zurückgegeben.
Dabei werden der DAT €uropa-Code®, der Marktindex, Bauzeitperiode, Benennungen und weitere Fahrzeuginformationen geliefert.
Damit sind alle, für eine notwendigen Daten für weitere Prozesse z.B. ermitteln der
technischen Daten, vorhanden.
