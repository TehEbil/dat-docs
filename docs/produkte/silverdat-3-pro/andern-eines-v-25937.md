# Ändern eines Vorgangs Neufahrzeug, Tageszulassung, Vorführfahrzeug

Um einen Vorgang für ein Neufahrzeug, eine Tageszulassung oder ein Vorführfahrzeug
zu ändern, können Sie die Funktion changeValuationN nutzen.

Generell gilt die Beschreibung zum [Verwalten von Vorgängen](../allgemeine-service/funktionsumfan-15005.md), ergänzt um die im Folgenden genannten Parameter.
