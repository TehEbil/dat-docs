# Alle Aufträge holen

Mit der Funktion getAllTask holen Sie alle aktuellen Aufträge und die dazugehörigen Auftragsinformationen. Es
gibt keine Eingabeparameter. Aufträge von der GUI können nicht mit dieser Methode
abgerufen werden.
