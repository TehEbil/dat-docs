# Verkaufsangaben holen

Mit der Funktion getSalesDetailsList holen Sie alle Verkaufsangaben zu der angegebenen Kundennummer.

Parameter getSalesDetailsList

| Name | Datentyp | Beschreibung | Schreibweise | Pflicht |
| --- | --- | --- | --- | --- |
| CustomerNumber | Integer | Kundennummer (Man verwendet hier nur die eigene Kundennummer ) | numerisch | x |
