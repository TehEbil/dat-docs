# Hereinnahmedaten holen

Mit der Funktion getAcceptanceDetailsList holen Sie alle angelegten Hereinnahmedaten zu der angegebenen Kundennummer.

Parameter getAcceptanceDetailsList

| Name | Datentyp | Beschreibung | Schreibweise | Pflicht |
| --- | --- | --- | --- | --- |
| CustomerNumber | Integer | Kundennummer (Man verwendet hier nur die eigene Kundennummer ) | numerisch | x |
