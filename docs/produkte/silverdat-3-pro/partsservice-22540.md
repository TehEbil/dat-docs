# PartsService

Die Soap Schnittstelle der SilverDAT 3 PRO bietet zusätzlich die Funktionen des Webservice PartsService zur Ermittlung der Ersatzteilinformationen und Fahrzeuginformationen.

Folgende Funktionen sind in PartsService enthalten:

- [getSparePartsDetails()](ermittlung-von-22533.md)
- [getSparePartsDetailsByVIN()](ermittlung-von-22547.md)
- [getSparePartsDetailsForDPN()](ermittlung-der-22556.md)
- [getSparePartsDetailsForDPNByVIN()](ermittlung-von-22563.md)

Die spezifischen Schnittstellenfunktionen der SilverDAT 3 PRO zum Verwalten der Kalkulationen und ihre Beschreibungen rufen Sie über folgende URL's
auf:

WSDL

PartsService

[https://www.dat.de/myClaim/soap/v2/VehicleRepairService?wsdl](https://www.dat.de/myClaim/soap/v2/VehicleRepairService)

Serviceaufruf

PartsService

[https://www.dat.de/myClaim/soap/v2/PartsService](https://www.dat.de/myClaim/soap/v2/VehicleRepairService)
