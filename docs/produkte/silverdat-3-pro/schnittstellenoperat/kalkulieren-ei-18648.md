# Kalkulieren eines Fahrzeugs ohne Speicherung

Die Funktion calculate() wird mit dem November Release im Jahr 2024 nach Einhaltung der Abkündigungszeit von
6 Monaten endgültig aus der Schnittstelle entfernt. Bitte stellen Sie rechtzeitig
auf die neue Funktion calculateN() um!

Mit der Funktion calculate() wird ein Fahrzeug in SilverDAT 3 PRO in Memory kalkuliert. Diese Funktion legt keinen Vorgang an.
