# Fahrzeugbeschreibung holen

Mit der Funktion getVehicleColorUpholstery holen Sie eine Fahrzeugbeschreibung zu einem bestehenden Vorgang.

Parameter getVehicleColorUpholstery

| Name | Datentyp | Beschreibung | Schreibweise | Pflicht |
| --- | --- | --- | --- | --- |
| DossierId | Long | Eindeutige Identifikationsnummer des Auftrags | numerisch | x |
