# Benutzerdefinierte Handelsspanne anlegen

###### Benutzerdefinierte Handelsspanne anlegen Mit der Funktion setValueInfluencingFactors kann die Handelsspanne für einen bestehenden Vorgang benutzerdefiniert angepasst werden. Rückgabe Im Erfolgsfall werden die wertbeeinflussenden Faktoren des entsprechenden Vorgangs zurückgegeben.
