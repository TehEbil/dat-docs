# Nutzungshinweise

Um die Webservices von autoglas Plus - SilverDAT inside oder SilverDAT calculateGlass nutzen zu können, benötigen Sie eine Lizenz und eine Freischaltung der zugehörigen
SOAP-Services. Bitte nehmen Sie Kontakt zum Vertrieb der DAT auf und erfragen Sie die Konditionen.

Sie können auch eine E-Mail an interfaces@dat.eu senden, mit der Bitte um Klärung der Nutzungsmodalitäten.

Wenn Sie versuchen, ohne eine gültige Zugangsberechtigung Zugriff auf unsere Webservices zu nehmen, wird Ihnen der Zugang verwehrt werden und Sie erhalten eine Fehlermeldung
"authorisationFailed".

Es handelt sich hier um SOAP-Services. In allen Schnittstellen werden sensible XML-Daten in lesbarer Form übertragen. Bitte benutzen Sie deshalb aus Sicherheitsgründen
spätestens beim produktiven Zugriff auf die Schnittstellen HTTPS anstelle von HTTP. Alle in diesem Dokument aufgeführten Internetaddressen sind auch über HTTPS ansprechbar.
