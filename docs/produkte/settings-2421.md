# Settings

| Attribut | Bedeutung | Typ und Wertebereich |
| --- | --- | --- |
| returnLanguage | Rückgabesprache; Übergabe nach ISO 639-1 (2 Buchstaben) | String |
| offset | Offset für Rückgabe, hiermit kann man dann im Resultset blättern… (z. B. erste Resultset-Seite = Offset 0, zweite Resultset-Seite = Offset 1 usw.);  -wenn offset leer, nur die ersten 100 Vorgänge zurückgeben (siehe numberOfContractsDisplayed);  -ist der offset größer als numberOfContractsDisplayed oder größer als die Gesamtausgabe, wird 100 zurückgegeben. | Integer |
| numberOfContractsReturned | max. 100, ansonsten Wert zwischen 1 und 100;  -ist Feld leer werden alle Vorgänge bis max. 100 zurückgegeben. | Integer |
