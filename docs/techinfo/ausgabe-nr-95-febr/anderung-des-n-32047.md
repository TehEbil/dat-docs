# Änderung des Namens vom complexType deleteAllManualPositionsResponse

Mit dem Februar Release 2026 wird in der VXS der complexType des Elementes deleteAllManualPositionsResponse von deleteAllManualPositionsResponse in deleteAllManualPositionsResponseType umbenannt. Bitte beachten Sie, dass das nur für das Produkt CalculateItalia gilt.

nötig, rechtzeitig entsprechende Anpassungen durch!
