# Erweiterung der Funktion exportDossierToDocument()

Mit dem September Release 2025 wird der Parameter printProduct der Funktion exportDossierToDocument() um folgende Werte erweitert:

| Name | Datentyp | Beschreibung |
| --- | --- | --- |
| maintenanceSchedule | String | Ermöglicht den Ausdruck eines Wartungsplans für ein Fahrzeug. |
| withoutInsuranceData | String | Ermöglicht den Ausdruck des Kalkulationsergebnisses ohne Versicherungsdaten. |
| withoutOwnerData | String | Ermöglicht den Ausdruck des Kalkulationsergebnisses ohne Daten des Fahrzeugeigentümers. |
| withoutDriverData | String | Ermöglicht den Ausdruck des Kalkulationsergebnisses ohne Fahrerdaten. |
| withoutGarageData | String | Ermöglicht den Ausdruck des Kalkulationsergebnisses ohne Werkstattdaten. |
| withoutOpponentData | String | Ermöglicht den Ausdruck des Kalkulationsergebnisses ohne Daten der Gegenpartei des Unfalls. |
| withoutEquipments | String | Ermöglicht den Ausdruck des Kalkulationsergebnisses ohne Ausstattungsdaten des Fahrzeugs. |
| withoutRepairCostExpansion | String | Ermöglicht den Ausdruck des Kalkulationsergebnisses ohne Reparaturkostenausweitung. |

Weitere Informationen finden Sie in unserem Kompendium unter: [exportDossierToDocument()](../../../produkte/silverdat-calculatepro-si/schnittstellenoperat/exportieren-vo-2338.md)
