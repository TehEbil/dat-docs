# Abkündigung der Funktion calculate()

Die Funktion calculate() wird mit dem Februar Release im Jahr 2026 nach Einhaltung der Abkündigungszeit von
6 Monaten aus der Schnittstelle endgültig entfernt. Bitte stellen Sie rechtzeitig
auf die neue Funktion calculateN() um!

Bitte berücksichtigen Sie dies in Ihrer eigenen Implementierung. Führen Sie, wenn
nötig, rechtzeitig entsprechende Anpassungen durch!
