# Wichtige Information für CalculationSummary

Wichtige Änderung!

Bitte beachten Sie, dass der Node CalculationSummary für die Zusammenfassung der Summenblöcke einer Reparaturkostenkalkulation, aus internen
Gründen weiterhin in den Schnittstellen enthalten ist und nicht entfernt wird. Verwenden
Sie zukünftig ausschließlich RepairCalculationSummary für die Auslesung der Summenblöcke.

Bitte berücksichtigen Sie dies in Ihrer eigenen Implementierung. Führen Sie, wenn
nötig, rechtzeitig entsprechende Anpassungen durch!
