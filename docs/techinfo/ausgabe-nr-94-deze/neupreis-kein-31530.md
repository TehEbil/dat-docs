# Neupreis kein Pflichtfeld mehr für Bewertung freier Aktenzeichen

Die Bewertung von freien Aktenzeichen ist jetzt auch möglich, ohne den Neupreis des
Fahrzeugs im Parameter OriginalPrice unter dem Node Valuation zu übergeben. Der Parameter BasePrice2 bleibt jedoch weiterhin eine Pflichtangabe.

Fahrzeuge lassen sich in freien Aktenzeichen also ab sofort auch über die Schnittstelle
ohne die Angabe eines Neupreises bewerten.
