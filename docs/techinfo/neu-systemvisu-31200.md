# Neu: Systemvisualisierung

Die DAT freut sich, ihren Kunden eine neue Funktion zur Verfügung zu stellen: die
Systemstatusanzeige. Dieses Tool ermöglicht es unseren Schnittstellenpartnern, jederzeit
über den Status unserer Anwendungen und Dienste informiert zu sein.

Wichtige Highlights:

- Echtzeit-Überwachung: Unsere Systeme werden kontinuierlich überwacht, sodass im Falle
  einer Störung sofortige Maßnahmen ergriffen werden, um Ausfallzeiten zu minimieren.
- Transparente Benachrichtigung: Ab sofort können unsere Partner über einen Webservice
  alle 5 Minuten über den Zustand der DAT-Systeme informiert werden.
- Ampelsystem: Der Status wird durch folgende Farben angezeigt:

  - Grün (Wert 1): Volle Verfügbarkeit
  - Gelb (Wert 0): Eingeschränkte Verfügbarkeit
  - Rot (Wert -1): Gestörte Verfügbarkeit
- Zusätzlich können allgemeine Nachrichten und spezifische Informationen zu den einzelnen
  Services abgerufen werden, sobald diese verfügbar sind.

Um den Systemstatus zu beauftragen, senden Sie bitte eine Mail an [interfaces@dat.eu](mailto:interfaces@dat.eu "Mail DAT interfaces") mit dem Betreff: Systemstatus

Weitere Informationen finden Sie im SilverDAT Schnittstellen Kompendium unter:

Allgemeine Informationen > DAT Developers Guide > Anzeige/Abfrage DAT Systemstatus
