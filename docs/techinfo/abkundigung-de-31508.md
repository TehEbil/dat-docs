# Abkündigung der Funktion getCalculationResults()

Die Funktion getCalculationResults() wird nach Einhaltung der Abkündigungszeit von 6 Monaten mit dem Februar Release im
Jahr 2026 aus der Schnittstelle entgültig entfernt. Bitte stellen Sie rechtzeitig auf die neue
Funktion getCalculationResultN() um!

Bitte berücksichtigen Sie dies in Ihrer eigenen Implementierung. Führen Sie, wenn
nötig, rechtzeitig entsprechende Anpassungen durch!
